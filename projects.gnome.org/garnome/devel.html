GARNOME Development

GARNOME is not maintained. JHBuild is the recommended build utility for the GNOME Desktop.

Bug Reports

Reporting bugs is a great way to help make the GNOME project a better desktop environment for new and experienced users alike. Very often problems are not fixed just because they are not known.

Patches

You can submit patches on the GARNOME mailing list or in Bugzilla. In the latter case remember to add the PATCH keyword to the bug, to bring it to the mantainers attention.

SVN

Here are some brief instructions how to checkout the latest code from SVN. For more detailed information, please see the Subversion and Subversion FAQ pages in the GNOME wiki.

You can also browse the GARNOME SVN Repository on the web through ViewCVS.

GARNOME Unstable (Development) branch

svn checkout http://svn.gnome.org/svn/garnome/trunk garnome-svn

GARNOME Stable branch

svn checkout http://svn.gnome.org/svn/garnome/branches/gnome-2-24 garnome-svn-2-24

Updating GARNOME

cd garnome-svn
svn update

IRC Discussions

Most of the development discussions surrounding GARNOME are done via IRC. If you have a question or comment regarding a future release, bug, or a request for enhancement, feel free to ask it in this channel.

Given IRC is a live, real time medium -- and the GARNOME developers occasionally like to sleep, anything typed in the channel will be logged by monkey.
