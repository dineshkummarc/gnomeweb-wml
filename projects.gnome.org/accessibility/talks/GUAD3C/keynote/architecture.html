ATK : in-process accessibility API, written to by apps and implemented on behalf of widgets by GAIL (gtk+, gnome-canvas), libgail-gnome (bonobo widgets), internal widget code (libzvt, gtkhtml2, others).
 
 

AT-SPI: out-of-process API used by AT: CORBA IDL definitions; libspi, CORBA/bonobo implementation; atk-bridge, exports ATK to AT-SPI; at-spi-registryd, central registration and event dispatch service.

Assistive Technologies, and AT services such as text-to-speech.
