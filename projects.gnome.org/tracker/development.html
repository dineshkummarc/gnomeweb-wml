Development

You can retrieve current development code with the following command:

git clone git://git.gnome.org/tracker

We have switched to git.

If you are new to git maybe there is a quick and dirty mail that you want to read.

cd into the root directory of the resulting sources and compile it:

cd tracker
./configure --prefix=/usr --sysconfdir=/etc --localstatedir=/var
make
sudo make install

Check the README for more information and other options.

For the GitWeb interface http://git.gnome.org/cgit/tracker/

Some ideas about the new Features that want to be added to Tracker can be found on the wiki.
