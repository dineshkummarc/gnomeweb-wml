Stable

Solang is under heavy development. There is a good chance that your GNU/Linux distribution will soon include Solang. Refer to your distribution's software installation instructions. You can also install Solang from source code. Note that you then need several development packages.

Unstable

Unstable versions are meant for developers and for those who want to try out the latest features and improve Solang. These versions is likely to have bugs. Please help us track them down.

Sources

Solang's source code can be found in GNOME Git, in the "solang" module. You can also browse them online. To get them, just execute

git clone git://git.gnome.org/solang
