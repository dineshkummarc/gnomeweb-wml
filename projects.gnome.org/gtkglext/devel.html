Source code

Releases are available from the Download page.

GtkGLExt's CVS repository can be browsed with ViewVC.

Contributing

GtkGLExt is maintained and developed on a volunteer basis. Contributions to improve GtkGLExt are welcome. Please consult the GtkGLExt mailing list and submit patches using GtkGLExt's SourceForge patch manager.

Mailing lists

  • gtkglext-list
    The gtkglext-list is a low-volume list that discusses use and development of GtkGLExt.
  • gtkglext-cvs
    The gtkglext-cvs list receives commit mail. It may be of interest to persons tracking the development of GtkGLExt.

Bug Reporting

Please report bugs using GtkGLExt's SourceForge bug tracker.
