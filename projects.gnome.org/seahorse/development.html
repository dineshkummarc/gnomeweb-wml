If you would like to volunteer to assist in development please subscribe and send an email to seahorse-devel. For detailed information on Seahorse development also check out the Seahorse Wiki for information including Seahorse architecture and progress on the various TODO items.

Bugs

A list of bugs and enhancement requests can be found bugzilla.

To receive bug and enhancement updates by email set your email settings in Bugzilla by add the user seahorse-maint@gnome.bugs.

There is now a --enable-debug option at compile time to give more detailed debugging information.

Git

The latest Seahorse source code is available for download at the GNOME repositories http://git.gnome.org/cgit/seahorse 
git clone git://git.gnome.org/seahorse
The source can be updated to the latest available at anytime. 
git pull --rebase
