What is Nautilus-Python?

Nautilus-python is a C extension for the Nautilus File Manager that provides access to the Nautilus extension API for python scripts. Using Nautilus-python, it is possible to do anything a C extension can do, but quicker and easier! Look below for a detailed list of features.

Compile it

The current released version of nautilus-python is 0.6.1, released January 19, 2010.

Download and extract the tarball, enter the program directory, then type:

$ ./configure $ make $ sudo make install (or run as root)

You can also fetch and compile the code from our git repository. To do so, type:

$ git clone git://git.gnome.org/nautilus-python

If compiling from a cloned git repository, simply replace the ./configure line with sh autogen.sh.

Dependencies

Nautilus-python depends on the following programs and libraries:

  • pygtk
  • pygobject
  • gnome-python
  • nautilus and libnautilus-extension
  • python

Make sure you have installed the development packages for these libraries before compiling nautilus-python.
