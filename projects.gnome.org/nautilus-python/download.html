Distribution Packages

Most distributions provide packages for nautilus-python and are available through their repositories. However, here are some links to various distributions' packages:

  • Debian GNU/Linux - apt-get install nautilus-python
  • Ubuntu - sudo apt-get install nautilus-python
  • Fedora / RHEL - yum install nautilus-python
  • OpenSUSE

Dependencies

Nautilus-python depends on the following programs and libraries:

  • pygobject >= 2.28.2
  • nautilus and libnautilus-extension >= 2.32
  • python >= 2.35

Make sure you have installed the development packages for these libraries before compiling nautilus-python.
