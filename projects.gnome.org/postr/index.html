postr

GNOME Flickr Uploader

Upload your photos to Flickr with Postr

Postr is a Flickr uploading tool for the GNOME desktop, which aims to be simple to use but exposing enough of Flickr to be useful. It isn't finished yet, but its a very capable uploader already.

The lastest version is 0.12.4 which was released on November 04, 2009 and it only fixes bugs of the series 0.12. Read the release notes. You may get the source code of Postr from GNOME servers.

Debian Lenny and Ubuntu Karmic provides packages for Postr 0.12.3.

Some Features

  • Simple user interface.
  • Set the common attributes for your photos: title, description, tags, sets, groups, privacy, licence, etc.
  • Nautilus integration: Either run postr when you are browsing your photos or just would to drag from Nautilus to drop them in Postr.
  • Session management: If you ran out of time, you may save your session and continue working on it later.

Contribute

Postr is a project with a small code base, hence it is more manageable than other big projects. If you are looking for a way to start contributing to Free/Libre/Open Source projects, or if you do not have enough spare time, Postr is a good project to start getting involved.

  • Filing bugs: Either run postr when you are browsing your photos or just would to drag from Nautilus to drop them in Postr.
  • Helping other people with their bug reports: You can help to clarify report's problems as well as confirming those bugs you can reproduce. We encourage you to read the Triage Guide, which was written with this purpose in mind.
  • Writing documentation: Postr does not have any documentation yet, but we would like to have a good one.
  • Writing code: Your contributions is welcomed. You may would like to start with those bugs tagged as gnome-love, which we think are easy to be approached by anyone (with or without experience on GNOME related framework).

It is written in Python using PyGTK and it is released under the GPL license version 2.

Installing from source code

To test Postr application is as easy as download the source code. For instance, if you have downloaded postr-0.12.4.tar.gz you should run: 

$ tar zxvf postr-0.12.4.tar.gz

$ cd postr-0.12.4
$ ./postr

And that is all!
