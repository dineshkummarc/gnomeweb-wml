Description of the project

LaTeXila is an Integrated LaTeX Environment for the GNOME desktop, written in Vala and covered by the GPL 3 license.

LaTeXila 2.0.8 is the latest stable release (June 14, 2011). Here is a summary of the functionalities:

  • Customizable buttons to compile, convert and view a document in one click
  • LaTeX commands completion
  • Symbol tables (Greek letters, arrows, etc)
  • Template managing: a few default templates, creating personal templates, etc
  • User-friendly compilation output (with Rubber)
  • Easy projects management
  • Some menus and toolbars with the principal LaTeX commands
  • File browser integrated
  • All common things found in a text editor...
  • Available in English, French, Spanish, Brazilian Portuguese, German and Italian

Development version

The project is hosted on this Git repository.
A repository is also available on GitHub, so you can fork it easily.

If you want to follow the development, you can download the source code from the Git repository:
$ git clone git://git.gnome.org/latexila

Later, to take the new commits you just have to do:
$ git pull

Roadmap

LaTeXila 2.2

  • Use Latexmk by default instead of Rubber for document compilation [done]
  • List of chapters, sections, etc to easily navigate in a document

LaTeXila ≥ 2.4

High priority

  • Check spelling
  • Improve LaTeX commands completion

Low priority

  • Insert an image easily
  • Insert a table from a spreadsheet (OpenOffice.org Calc, Gnumeric, ...) easily
  • Create bibliography items with BibTeX easily
  • Forward and backward search: find position in DVI or PDF from the *.tex file, and inversely (SyncTeX)
  • Preview a selected part of a document
  • Possibility to edit shortcuts
  • Incremental search optimization
  • More documentation
  • XeTeX, LuaTeX and Omega support
  • Plain TeX and ConTeXt support
  • Code folding
  • ...

LaTeXila 3.0

I began LaTeXila as an independent application instead of a Gedit plugin mainly because of the lack of the profiles feature. See this mail for a more in-depth explanation.
Gedit will support profiles in the future as a way like it is explained in the mail. So when it'll be done, I consider transforming LaTeXila as a Gedit plugin. It'll be a big change in the code structure and also a big change for users (as a good enhancement), so the version will be 3.0.

Design

If you want to improve the logo, the application icon or the other icons, let me know ;)

Translations

If you want to make a new translation, read the instructions in the file HACKING. Contact me if you need some help.

Hacking

If you are interested to contribute to the code of the project, contact-me and tell me what you want to do (see the file TODO or the Roadmap). If you have no idea of what to do, I can obviously give you some interesting work. I can help you if you don't understand all the code. Also, tell me if you see some possible improvements.
