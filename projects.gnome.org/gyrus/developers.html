Contributing to gyrus development

Getting the sources

Gyrus development takes place in the GNOME git server. You can check out the latest code using the git program like this:

$ git clone git://git.gnome.org/gyrus

You can also browse the cgit web interface.

Bugs, patches, requests, and Bugzilla

We prefer to use GNOME Bugzilla to keep track of requests, patches, bugs, and features which are being worked on. Before submitting a bug report, please check it's not already reported.

A general overview of the current status of the project bugs can be found in the gyrus product page.


Translating

Just as many other GNOME applications, gyrus is translated by the GNOME translation team.


Documentation

If you want to write documentation for gyrus, you should join the GNOME Documenation team.


Contributing to the GNOME project

More information about getting involved with other GNOME apps (and many tips that apply to gyrus) can be found at http://live.gnome.org/JoinGnome


Downloads

The latest releases can always be found at the GNOME ftp site.

The latest release is 0.3.9 (announcement).


Feedback

Bugs should be reported to the GNOME Bug Tracking System.

You can get in touch with developers and other users by sending a mail to the gyrus mailing list [Archives].
