Contribute

Test

Download gnome-scan from Gnome SVN, install depencies (especially Sane, Gegl and Babl) and test Gnome Scan. Report bugs to Gnome's bugzilla. Also, if you have some nice ideas for features, interfaces, and more, don't hesitate to told us them on the bugzilla !

Translate

Like Gnome, Gnome Scan take care of being accessible in many languages. Don't hesitate to translate, it's not huge. Download latest translation template and report your translation. Ask Étienne Bersac if you want to translate the web site.

Develop

Gnome Scan is only a piece of the software stack between scanner and users. Gnome Scan is certainly the most visible part, however, to improve it, we must improve all pieces of the stack. Softwares like SANE and HAL are key projects for Gnome Scan. Also, work downstream in the image processing chain is very very important. Integrate image processing software in the Gegl pipeline would help Gnome Scan adding much wanted features.

As soon as possible, write scan plug-in for your (prefered) app. Take the advantage of scanner. Beginning with Gnome Scan 0.5, you'll be able to run a virtual "from files" scanner in order to test your software without hardware access. Reporting lake of documentation and bugs will help.

Distribute

Package Gnome Scan for you (favorite) distribution and submit it for inclusion may save a lot of lost end users. Follow the Linus law : "Given enough eyeballs, all bugs are shallow" !

Thank you !
