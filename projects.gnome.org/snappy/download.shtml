Stable

Development

Development version is meant for developers and for those who want to try out the latest features and improvements in snappy. This version is likely to have bugs. Please help us to track them down.

Source code

snappy sources can be found in GNOME Git, in the "snappy" module. Also, you can browse the latest sources online. To get them, just run:

git clone git://git.gnome.org/snappy
