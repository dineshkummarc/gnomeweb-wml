Add-ins

Add features

Find out which add-ins are already installed to your Tomboy from the Preferences window (tip: right click on the Tomboy icon to open context menu).

You can add new features and to Tomboy easily by using add-ins. There are few add-ins already included in the default installation to be enabled if you want to. Just right click on the Tomboy icon and open preferences and select the add-ins tab. More add-ins can be found from Tomboy wiki's add-ins list page. New add-ins can be added simply by drag and drop to the add-ins folder.

Top add-ins

Backlinks

See which notes link to the one you're currently viewing.

Bugzilla

Allows you to drag a Bugzilla URL from your browser directly into a tomboy note. The bug number is inserted as a link with a little bug icon next to it.

Evolution

Allows you to drag an email from Evolution into a tomboy note. The message subject is added as a link in the note.

Export To HTML

Exports individual notes to HTML.

Fixed Width

Adds fixed-width font style.

Note of The Day

Automatically creates a "Today" note for jotting down daily thoughts.

Sticky Notes Importer

Import your notes from the Sticky Notes applet.
