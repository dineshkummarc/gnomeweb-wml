Grab the Source from Git

Paul Smith wrote a very useful Makefile that automates most of the process of building Evolution source code from the GNOME Git repository. See Paul's article "Build Evolution from Source Made Simple" for full details.

You can also use GNOME's default tool Jhbuild for building Evolution from Git. For general information how to use GNOME Git see the corresponding wikipage.

Microsoft Windows Packages

An experimental build of Evolution 3.0.2 is available for Microsoft Windows. Please read this blog post for how to install these RPMs on Windows.
