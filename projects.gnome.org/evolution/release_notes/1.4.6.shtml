Evolution 1.4.6 Release Notes

Ximian Evolution 1.4.6 is now available. This update includes bug fixes as a result of community and customer feedback received since version 1.4.5. This release also includes an update for Ximian Connector.

Supported Systems

Evolution 1.4.6 is supported on the following operating systems:

  • Mandrake Linux 9.1
  • Red Hat Linux 7.3, 8.0 and 9
  • Suse Linux 8.2, 9, and Suse Linux Desktop 1.0
  • Solaris 8
