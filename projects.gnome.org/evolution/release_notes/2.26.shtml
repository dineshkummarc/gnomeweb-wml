Evolution 2.26 Release Notes

The Evolution Team is pleased to announce the release of Evolution 2.26 and Evolution-Data-Server 2.26.

What's New in Evolution 2.26?

  • Evolution-MAPI - Connect to Exchange 2007 servers
  • Import Personal Storage Table (.pst) files from Microsoft Outlook
  • Task and Memo support for CalDAV
  • Dropped our libical fork and merged with upstream libical
  • Calendar performance improvements

Also, nearly 400 bugs and 30 crashers fixed.

Thanks to all who contributed to the Evolution 2.26 release.

Reporting Bugs

If you have problems with Evolution 2.26, please take the time to submit the bug using Bug Buddy or at http://bugzilla.gnome.org/. Try to fill in as much detail as you can regarding the circumstances that lead to the problem.

Kindly check if your bug has been reported before by using the search functionality of Bugzilla.
