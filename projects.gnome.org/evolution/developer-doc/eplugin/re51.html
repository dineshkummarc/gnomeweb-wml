Constants

EM_POPUP_FOLDER_FOLDER

EM_POPUP_FOLDER_STORE

This is a store.

EM_POPUP_FOLDER_INFERIORS

This folder may have child folders.

EM_POPUP_FOLDER_DELETE

This folder can be deleted or renamed.

EM_POPUP_FOLDER_SELECT

This folder exists and can be selected or opened.
