e_popup_add_items
Prev Chapter 9.  EPopup  Next

Name

e_popup_add_items —

Synopsis

void e_popup_add_items (emp,  
 items,  
 freefunc,  
 data); 
EPopup *  emp;
GSList *  items;
EPopupItemsFunc  freefunc;
void *  data;

Arguments

emp

An EPopup derived object.

items

A list of EPopupItem's to add to the current popup menu.

freefunc

A function which will be called when the items are no longer needed.

data

user-data passed to freefunc, and passed to all activate methods.

Description

Add new EPopupItems to the menus. Any with the same path will override previously defined menu items, at menu building time. This may be called any number of times before the menu is built to create a complex heirarchy of menus.
