Name

em_format_class_add_handler —

Synopsis

void em_format_class_add_handler (emfc,  
 info); 
EMFormatClass *  emfc;
EMFormatHandler *  info;

Arguments

emfc

EMFormatClass

info

Callback information.

Description

Add a mime type handler to this class. This is only used by implementing classes. The info.old pointer will automatically be setup to point to the old hanlder if one was already set. This can be used for overrides a fallback.

When a mime type described by info is encountered, the callback will be invoked. Note that info may be extended by sub-classes if they require additional context information.

Use a mime type of “foo/ *” to insert a fallback handler for type “foo”.
