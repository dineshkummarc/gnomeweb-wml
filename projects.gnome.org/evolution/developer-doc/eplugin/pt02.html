Evolution Hook Points.

This section enumerates all of the published hook points and target types available in each component in Evolution.

Table Format

IdThe hook point id.
TargetThe target which this hook uses for its context data. Targets are described in a following section.
ItemsIf appropriate and defined, specifies identifying path names of items which make up the hook. e.g. popup menu items, and configuration pages. These item specifications allow the plugin writer to position their items appropriately.
