Reporting Bugs

If you have problems with Evolution, please take the time to submit a report using Bug Buddy or at http://bugzilla.gnome.org/. Try to fill in as much detail as you can regarding the circumstances that lead to the problem.

Kindly check if your bug has been reported before by using the search functionality of Bugzilla.
