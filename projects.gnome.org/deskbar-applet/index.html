About Deskbar-Applet

The goal of Deskbar-Applet is to provide an omnipresent versatile search interface. By typing search terms into the deskbar entry in your panel you are presented with the search results as you type.

Searches are handled by a series of plugins. Deskbar-Applet provides a simple interface to manage these plugins to provide you with the search results that fit your needs.

A related idea was published earlier as Auric Applet. Similar software from the GNOME family includes contact-lookup-applet, mini-commander applet (in gnome-applets), webeyes and GNOME Launch Box and the Alt-f2 dialog.

Feature highlights

Generic

  • Sits in the panel waiting for input
  • Global configurable hotkey to directly focus the entry and start typing
  • Pluggable backend architecture, allows to add custom backends in home/system dir. Allows third parties to add a backend
  • Sorting of backend results order is adaptable easily.
  • Intuitive history menu, to repeat frequent actions
  • Translated in 69 languages
  • Use themed icons associated with actions, to have a visual feedback

Backends

  • Beagle Live: Search all of your documents (using Beagle), as you type
  • Calculator: Calculate simple equations
  • Computer Actions: Logoff, shutdown, restart, suspend and related actions
  • del.icio.us Bookmarks: Search your del.icio.us bookmarks by tag name
  • Developer Documentation: Search Devhelp for a function name
  • Dictionary: Look up word definitions in the dictionary
  • Files and Folders Search: Find files and folders by searching for a name pattern
  • Files, Folders and Places: View your files, folders, bookmarks, drives, network places by name
  • Google Code Search: Search public source code for function definitions and sample code
  • Google Search: Search Google directly from Deskbar
  • History: Recognize previously used searches
  • OpenSearch: Search any OpenSearch compliant website
  • Programs: Launch a program by its name and/or description
  • Recent Documents: Retrieve your recently accessed files and locations
  • Tomboy Notes: Search your Tomboy notes
  • Twitter and identi.ca: Post updates to your Twitter or identi.ca account
  • Web: Open web pages and send emails by typing a complete address
  • Web Bookmarks: Open your web bookmarks by name (Mozilla, Firefox 2, epiphany)
  • Web History: Open your web history by name
  • Web Searches: Search the web via your browser's search settings
  • Window Switcher: Switch to an existing window by name
  • Wikipedia Suggest: As you type, Wikipedia will offer suggestions
  • Yahoo! Search: Search Yahoo! as you type
  • Yahoo! Suggestions: Provides suggestions for queries related to the submitted query
