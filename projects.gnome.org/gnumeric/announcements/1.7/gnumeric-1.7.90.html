Gnumeric 1.7.90 aka "Release Candidate 1" is now available.

This is the first release candidate leading up to 1.8.

In this release, we fixed more of the long-standing crashes that Sum1 tirelessly has dug up. Also, Andreas has been polishing printing support.

This release requires the concurrently released Goffice 0.5.2, see below.
