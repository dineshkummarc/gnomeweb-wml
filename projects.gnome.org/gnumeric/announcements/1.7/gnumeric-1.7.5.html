Gnumeric 1.7.5 aka "Testing, testing, 1-2-3, testing" is now available.

This release requires the concurrently released libgoffice 0.3.4.

The major news in this release is that Gnumeric's test suite has been automated as much as possible. This is aimed at preventing further brown-bag releases.

The tests do not cover all of Gnumeric yet. In particular, we have not yet found a good way to automate the testing of the GUI parts.

Note. that the new "SAX" importer for .gnumeric files has been turned on by default. Please report any problems so we can fix them and construct tests for the future. We already know that sheet objects have some problems.
