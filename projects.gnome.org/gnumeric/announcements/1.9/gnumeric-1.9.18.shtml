Gnumeric 1.9.18 aka "TBD" is now available.

The Gnumeric Team is pleased to announce the availability of Gnumeric version 1.9.18. This is a bug-fix release in preparation for 1.10.

  • Andreas
    • Add paste special flip horizontally and vertically. [Bug 393367].
    • Add EIGEN function to calculate eigenvalues and eigenvectors of real symmetric matrices.
    • Add principal components analysis.
    • Load graph series labels and categories from ODF files. [Bug 379220].
    • Write series labels to ODF files.
    • Fix tab behaviour. [Bug 388344].
    • Print sheet widget objects. [Bug 144787].
    • Allow sheet object size and position to be specified via a dialog.
    • Allow the sheet object name to be specified. [Bug 596545].
    • Add menu item to select sheet objects. [Bug 338615].
    • Enable sheet tab menu when objects are selected. [Bug 604496].
    • Fix empty cell handling of the advanced filter. [Bug 599064].
    • Fix scroll bar problem with large frozen panes. [Bug 68603].
    • Singleton selections now cause the sort buttons to sort the whole sheet below the selection. [Bug 141313].
    • Have the formula guru auto-quote unparsable expressions. [Bug 442941].
    • Create default tip for hyperlinks. [Bug 316974].
    • Add GET.LINK. [Bug 128534].
    • Add "Don't change formulae" to paste special. [Bug 123326].
    • Select pasted objects. [Bug 604227].
    • Add auto expression menu items to add formula. [#8872].
    • Some updates to the Gnumeric manual.
    • Fix tool tip position for recent gtk (2.18 or newer). [Bug 605723].
    • Fix NPV documentation. [Bug 22381].
    • Fix type errors. [Bug 606066].
    • Add surface plot description. [Bug 604285].
    • Include the short functions descriptions in the documentation. [Bug 606354].
    • Fix sheet handling in validation setup. [Bug 606458].
    • Allow some of the text export properties to be saved as default. [Bug 559749].
    • Fix printing of repeated rows and columns. [Bug 607451].
    • Fix import export of line type in scatter plots. [Bug 605043].
    • Force sharing of first series in line plots when requested. [Bug 605330].
    • Fixed mouse cursor for object resizing in RTL mode. [Bug 605514].
  • Jody
    • Fix use of ExprEntry from glade on win32.
    • More win32 build fixes (xslt for libgda & poppler).
    • Fix pango test harness.
    • Fix use of Tab in ExprEntry for gurus.
    • Adapt to latest gtk-2.19.3 GSEAL.
  • Morten
    • Fix GCD border case. [Bug 606870].
