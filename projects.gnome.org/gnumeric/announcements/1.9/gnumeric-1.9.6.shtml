Gnumeric 1.9.6 aka "TBD" is now available.

The Gnumeric Team is pleased to announce the availability of Gnumeric version 1.9.6. This release requires the concurrently released goffice 0.7.5. We also recommend libgsf 1.4.12.

In this release some major items stand out:

  • Variable sheet size. It is not possible to change the number of rows and columns in a sheet. This feature is still experimental. [Jean, Morten]
  • Sheet objects got some much-needed attention. [Andreas]

In addition, the usual collection of bugs were fixed.

Attention packagers: goffice no longer depends on libgnome, libgnomeui, and gnome-vfs. (To get the full benefit of this, make sure libgsf is not compiled to require gnome-vfs and bonobo.)

Attention packagers: please do not ship goffice with equation enabled.
