Gnumeric 1.10.7 aka "TBD" is now available.

The Gnumeric Team is pleased to announce the availability of Gnumeric version 1.10.7. For best results, use with the concurrently released Goffice 0.8.7.

This is a bug-fix release. Andreas has been continuing his quest to drive the bugzilla bug count down. We are now down more than 50% from when he started.

  • Andreas
    • Improve function syntax and formula guru tooltips.
    • Avoid superfluous redo items under validation failure. [Bug 622509].
    • Fix validation for array and range entry. [Bug 622511].
    • Autofit column widths in stf import. [Bug 427382].
    • Ensure the table range is unlocked before allowing the data table to be configured.
    • Also store any markup if entering text in several cells at once. [Bug 622530].
    • Read HTML files with nested tables. [Bug 594789].
  • Jean
    • Fixed autofilter (and other) combo positions. [Bug 621817].
  • Morten
    • Improve function syntax tooltips.
    • Fix excessive memory use problem with xlsx. [Bug 622365].
    • Make BackSpace work as Delete on pane. [Bug 622599].
    • Improve handling of MS-style HTML in clipboard. [Bug 617801].
