Graphical Plots

  • The Gnumeric Manual, version 1.8
  • Graphics: Plots, Images, Widgets, and Drawings

Gnumeric includes a powerful mechanism to create graphical plots which present visually the data contained in a worksheet.

The first sub-section of this section starts with an overview of graph creation, discusses terminology, and presents graph components and their organization. The next sub-section explains the "Graph Guru" which is the tool used to configure graphs. The subsequent sub-section presents the different types of plots which can be included in a graph and the final sub-section explains how data can be pre-selected to speed up the process of graph creation.

Graphs are exceedingly effective communication devices. Unfortunately, this means that one cannot determine, simply based on the data to be plotted, what type of plot will be the most effective. Instead users must familiarize themselves with the various types of plots which are available and decide for themselves which plot type is the most effective to communicate an idea. Section 7.2.3 ― Plot Types contains a detailed explanation of the plot types available in Gnumeric.
