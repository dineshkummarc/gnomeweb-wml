File Formats

  • The Gnumeric Manual, version 1.8
  • Working with Files

Gnumeric supports numerous file formats in addition to its own XML based format. The table below lists the formats currently supported. In the table below, the name of the format is a link which can be used to jump to the section which discusses the format.

Table 9-1 The file formats supported by Gnumeric
Format Extension Open Save
Gnumeric .gnumeric / .gnm YES YES
Applix .as YES
Comma Separated Values .csv YES YES
Data Interchange Format .dif YES YES
GNU Oleo .oleo YES
HTML .html / .htm YES YES
LaTeX .tex YES
Linear and Integer Program none / .mps YES
Lotus 1-2-3 .wk1 / .wks YES
Microsoft Excel .xls YES YES
Multiplan SYLK none / .slk YES
OpenOffice.Org / StarOffice .sxc YES YES
Plan Perfect .pln YES
Postscript .ps / .eps YES
PDF .pdf YES
Quattro Pro .wb1 / .wb2 / .wb3 YES
SC/XSpread none / .sc YES
Tab Separated Values .tsv / .tab YES YES
Text Formats .txt / .text YES YES
TROFF .me YES
Xbase .dbf YES
XHTML .xhtml / .html YES
