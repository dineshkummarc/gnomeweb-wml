Synopsis

COMPLEX(x,y,i)

Arguments

x: real part

y: imaginary part

i: the suffix for the complex number, either "i" or "j"; defaults to "i".

Note

If i is neither "i" nor "j", COMPLEX returns #VALUE!

Microsoft Excel Compatibility

This function is Excel compatible.
