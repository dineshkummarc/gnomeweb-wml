Synopsis

IMSECH(inumber)

Description

IMSECH returns the complex hyperbolic secant of the complex number z (inumber), where

sech(z) = 1/cosh(z).

  • If inumber is not a valid complex number, IMSECH returns #VALUE! error.

Examples

IMSECH("1+j") equals 0.498337-0.5910838j.

See also

IMCSCH, IMCOTH.
