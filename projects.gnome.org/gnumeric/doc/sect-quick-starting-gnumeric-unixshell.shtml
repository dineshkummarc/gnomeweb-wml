Starting Gnumeric from a *NIX terminal

  • The Gnumeric Manual, version 1.10
  • A Quick Introduction
  • Starting Gnumeric the First Time

You can also start Gnumeric from the command line in an xterm window or equivalent terminal emulator. Open a terminal, and, at the shell prompt type: 

gnumeric &
which will start the program and send it into the background.

You must be running the X window system to run Gnumeric.
