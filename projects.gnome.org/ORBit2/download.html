Download

Most distributions of GNU/Linux and BSD are shipped with pre-compiled stable release of ORBit2. To update ORBit2 on your host, have a look at web side of your favored distributor.

If you are familiar with automake tools, feel free to fetch source code of latest stable release offered by GNOME for download. Or, to join ORBit2 project, fetch development branch from SVN and compile as described below. ORBit2 requieres at least glib-2.0 or later release. Please ensure it is is available on your host.

Joining Development

The shared code resides in GNOME Git repository (Git howto). To check out the latest version:

git clone http://git.gnome.org/git/ORBit2

If installing to /opt set environment variables as follows:
export PKG_CONFIG_PATH=/opt/lib/pkgconfig
export GNOME2_DIR=/opt
export LD_LIBRARY_PATH=/opt/lib:${LIBRARY_PATH}
export PATH=/opt/bin:${PATH}

Now enter ORBit2 directory, compile and install ORBit2:
cd ORBit2; ./autogen.sh --prefix=/opt; make; make install
