Stable

No stable version yet. Take a look at development versions. You can install GNOME Nanny from source code.

Development

Development version is meant for developers and for those who want to try out the latest features and improve Nanny. This version is likely to have bugs. Please help us to track'em down.

From the bakery

Nanny sources can be found in GNOME Git, in the "nanny" module. Also, you can browse the latest sources online. To get them, just execute

git clone git://git.gnome.org/nanny
