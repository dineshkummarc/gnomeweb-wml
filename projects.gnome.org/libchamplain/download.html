Stable

Stable version is recommended for most users. There is a good chance that your Linux distribution already includes libchamplain. Refer to your distribution's software installation instructions.

You can also install libchamplain from source code. Note that in this case you need several development packages listed on the libchamplain's home page.

Development

Development version is meant for developers and for those who want to try out the latest features and improve libchamplain. This version is likely to have bugs. Please help us to track them down.

From the bakery

libchamplain sources can be found in the libchamplain repository of git.gnome.org. Also, you can browse the latest sources online. To get them, just execute

git clone git://git.gnome.org/libchamplain

We also use gitorious.org for tracking and merging patches from external contributors. See our project page at gitorious. Consider creating an account there if you want to contribute your code.
