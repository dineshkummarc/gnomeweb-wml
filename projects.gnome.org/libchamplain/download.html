Stable

Stable version is recommended for most of the users. There are more and more chance that your Linux distribution includes libchamplain. Refer to your distribution's software installation instructions. You can also install libchamplain from source code. Note that you then need several development packages.

Development

Development version is meant for developers and for those who want to try out the latest features and improve libchamplain. This version is likely to have bugs. Please help us to track them down.

From the bakery

libchamplain sources can be found in Gitorious, in the libchamplain project. Also, you can browse the latest sources online. To get them, just execute

git clone git://gitorious.org/libchamplain/mainline.git
