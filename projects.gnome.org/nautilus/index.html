A powerful file manager


See Nautilus in action

Nautilus is a graphical shell for the GNOME desktop enviornment that makes it easy to manage your files and the rest of your system.

These pages are meant to help you find information about Nautilus.


We want you to be part of the fun

Are you a coder, artist, documentation writer or translator? Join the fun!


Nautilus is released under the GNU General Public License (GPL).


Downloads

Nautilus should come bundled with your distribution.

The latest releases can always be found at the GNOME ftp site.

The latest stable release is 2.24.0 (changelog).

Make sure to get the latest release of eel, the extra library that Nautilus depends on.


Feedback

Bugs should be reported under the Nautilus product of the GNOME Bug Tracking System.

You can get in touch with developers and other users by sending a mail to the nautilus mailing list [Archives] or by joining the #nautilus channel on irc.gnome.org.
