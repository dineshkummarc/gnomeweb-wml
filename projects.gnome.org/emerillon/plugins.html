Emerillon can be extended with plugins. Check the Wiki or the Developers page for instructions on how to develop new ones.

Plugins shipped with Emerillon

Some plugins are shipped with Emerillon and form its basic features.
  • Placemarks: Save places and zoom levels.
  • Search: Search the map using geonames.org web services.
  • Copy Link: Copy a link to online map services to the current position.
  • Map Position: Displays the position at the middle of the map in the statusbar.
Check this Plugins Wiki page for a list of plugins ideas.
