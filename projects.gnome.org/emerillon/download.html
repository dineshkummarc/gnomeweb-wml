Stable

Stable version is recommended for most of the users. There are more and more chance that your Linux distribution includes Emerillon. Refer to your distribution's software installation instructions.

You can also install Emerillon from source code. Note that you then need several development packages.

No stable release has been made yet.

Development

Development version is meant for developers and for those who want to try out the latest features and improve Emerillon. This version is likely to have bugs. Please help us to track them down.

From the bakery

Emerillon sources can be found in git.gnome.org, in the Emerillon project. Also, you can browse the latest sources online. To get them, just execute

git clone git://git.gnome.org/emerillon
