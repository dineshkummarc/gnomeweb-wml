What is Emerillon?

Emerillon is a map viewer. Aiming at simple user interface, Emerillon is a powerful, extensible application. It is pronounced Ey-may-ree-yon.

It features OpenStreetMap based maps: the street map, the cycling map and the transportation map. Use it to:

  • Browse maps,
  • Search the map for places,
  • Placemark places for later quick access.

Emerillon is named after the Émérillon, one of the three boats that visited New France under Jacques Cartier's command in 1535. Émérillon is also French for Merlin Falcons.
