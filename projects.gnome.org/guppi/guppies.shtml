Other Stuff Named "Guppi"

  • Guppi is a children's clothing retailer that operates throughout Sweden.
  • Guppi is also a non-free software product that is, to quote the web page, "a sophisticated web-based publishing system for online periodical publications".
  • GuPPI is a hardware gadget, developed at MIT, for doing signal processing.
  • Guppi also seems to be someone's nickname.
  • Dave Thompson has registered guppi.org for his home page, which offers no explanation for the domain name.

There are many more such examples, many more than I would have expected for a misspelling of an uncommon word.
