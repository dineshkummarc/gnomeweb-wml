Packages

It is advisable to try the pre-compiled package for your distribution before you try to make GNOME Power Manager from source.

SVN

You can check out the latest GNOME Power Manager code using SVN. You can use the viewcvs interface too. 

svn checkout http://svn.gnome.org/svn/gnome-power-manager/trunk/ gnome-power-manager-trunk

Compiling The Code

To compile and make, use the following procedure: 

cd gnome-power-manager
./autogen.sh --prefix=/home/hughsie/root --enable-applets --with-gconf-source=xml::/home/hughsie/.gconf
make
make install

Downloads

GNOME Power Manager should come bundled with most modern distributions.

The latest source releases can always be found at the GNOME ftp site.

Make sure to get latest release of HAL, the service that GNOME Power Manager depends on.


Feedback

Bugs should be reported to the GNOME Bug Tracking System.

Please read the FAQ and the bug reporting guidelines before creating a new bug.

You can get in touch with developers and other users by sending a mail to the Power Manager mailing list or search the [archives].
