Building from source

While this can take a while, it's really the best way to get the latest Gnome/OO.o integration work running for you. The main build and associated documentation can be found here, the build instructions are familiar (with a first-time twist):

./configure
make
make install

NB. strange prefixes (not /usr) are unlikely to work. Also, while this makes an excellent first build, it's not a good way to hack on G/OO.o for more information on that see the hackers guide.
