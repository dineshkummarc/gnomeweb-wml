Characteristics of the management software

Printable version
ChronoJump allows to register any jump with the repetitions that is desired, initiating in or outside the platform, registering independently times of contact and flight times, as well as extra load, height of fall, jumps or time limitation and descriptions of technique.

ChronoJump uses a small data base Sqlite that enables to tie jumps to subjects and sessions, and to make all type of comparisons between these events.

It is possible and simple to create new types of jumps with the described variables previously.

It is multiplatform; being programmed in C# using Mono, it would have to work correctly in GNU/Linux, MacOSX, Windows... until the moment works in GNU/Linux and Windows.

It incorporates the following statistics (whenever the opposite does not specify itself, it is allowed to show averages, maximum values, or all the values of only one session or between an indetermine number of sessions (longitudinal), and with distinction of sex)

  • Summary of one or several sessions
  • Study of an only subject
  • Flight time of simple jumps
  • Contact time, flight time and index between both event, in jumps with fall and in repetitive jumps (shortly will be also implemented the takeoff speed).
  • Power in repetitive jumps
  • Evolution of repetitive jumps (only in one session)
  • Index of elasticity and index of arms use
  • ... (ChronoJump modularity makes easier the inclusion of new statistics)
Chronojump allow to record also races, reaction time and rhythm.

Graphics of all the statistical described by means of the graphical library NPlot.

Export jumps to spreadsheets (CSV format).

Create personalized reports of all data and statistics (HTML format).

Characteristics of the timekeeper
The Chronopic timekeeper is based on the trainning card Skypic (consult the documentation), and offers a precision of one millisecond according to what has been able to verify. [Pending to detail how has been verified].

Some jump timekeeper systems use only a personal computer connected directly to the platform. The modern computers are multitask, characteristic that prevents to guarantee a trustworthy measurement because they can be making any other task in a given moment. More info in the Hardware section.

License

ChronoJump is free software  distributed under the GPL license. The program and chronometer source code is available, it is also possible to modify and redistribute.

The chronometer and the contacts platform come along with construction guidelines.

It is not possible to verify the validity and fiability of a digital measuring instrument if the source code is not available. Therefore Chronojump is appopriate to investigations.
