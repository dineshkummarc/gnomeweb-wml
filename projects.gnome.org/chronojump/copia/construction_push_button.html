Construction of a push button


Xavier de Blas Foix, Josep Ma Padullés. Chronojump project. April 2009.
Description
We build a push button to be connected to the Chronopic.
Materials
  • 1 Push button (not interruptor), normally opened
  • 1 Little box
  • 1 RCA male connector
  • sandpaper
  • Soldering iron
Instructions
  1. Materials
  2. Sanding the box
  3. Connecting cables to the push button
  4. Connecting cable to RCA male
  5. Push button done
