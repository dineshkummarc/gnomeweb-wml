GNOME Network (http://www.gnome.org/projects/gnome-network/) is a set of
client network-oriented tools, which currently contains a network
information tool, a remote shell and desktop clients, and a personal web
server.

Changes since 1.99.1
--------------------

  General:

    - New website at http://www.gnome.org/projects/gnome-network/
(Carlos García)

  Network information tool:

    - More IPv6 support (Carlos García)

  Remote desktop client:

    - Replaced with tsclient (Erick Woods, Rodrigo Moya)
    - Some HIG-ification (Erick Woods, Rodney Dawes)

  Remote shell client:

    - Fixed gnome-terminal argument passing (Emil Soleyman-Zomalan)
    - Added IPv6 support (Carlos García)

  Personal Web Server:

    - Added personal web server tool (Álvaro Peña)

  Translations:

    - Azerbaijani (Metin Amiroff)
    - Czech (Miroslav Trmac)
    - Dutch (Vincent van Adrighem)
    - Portuguese (Duarte Loreto)
    - Serbian (Danilo Å egan)
    - Spanish (Pablo Gonzalo del Campo)
    - Swedish (Christian Rose)
    - Vietnamese (Pablo Saratxaga)
    - Walloon (Pablo Saratxaga)
