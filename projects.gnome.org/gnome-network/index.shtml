About GNOME Network

GNOME Network is a set of client network-oriented tools, which currently contains a network information tool, a remote shell and desktop clients and a personal web server and a desktop sharing tool.

GNOME Network is Free Software distributed under the terms of the GNU GPL

GNOME Network Mailing List

The GNOME Network mailing list can be reached at gnome-network-list@gnome.org. If you want to subscribe to the mailing list, send mail to: gnome-network-list-request@gnome.org and in the body of your message put the word "subscribe". The lists are also archived.
