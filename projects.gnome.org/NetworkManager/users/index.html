Using NetworkManager

Once the NetworkManager service is started, and the applet is running, using NetworkManager is a piece of cake. For information on how to get the service started, see the administrators page. NetworkManager will scan for available wireless networks. To view the list of available networks, simply click on the applet.

NetworkManager will automatically connect to networks it knows about, but you will need to manually connect to a network at least once. Simply select the network from the list, and NetworkManager will automatically attempt to connect. If the network requires an encryption key, NetworkManager will prompt for one. Once the correct key is input, the connection will be made.

To switch wireless networks, simply choose another one from the applet's menu.

Supported Wireless Hardware

A list of the wireless cards and drivers that NetworkManager interoperates with can be found here: http://live.gnome.org/NetworkManagerHardware.
