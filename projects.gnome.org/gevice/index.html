Gevice: GNOME Network Device Manager

Gevice is a tool for the administration of network device.

Features

The main features gevice includes are:
  • Allow you to draw the net model.
  • Allow you to print the net diagram in png,gif,jpg formats.
  • Allow you to save your model in database (Postgresql)
  • Allow you to manager the device, in terminal (telnet,ssh or local)
  • Allow export data to CSV file. Use "|" to separate fields (or other).

Download

Gevice development takes place in the GNOME GIT Server. You can check out the latest code using the git program like this:

$ git clone git://git.gnome.org/gevice

The latest releases can always be found at the GNOME ftp site.

The latest release is 0.5.2
