Introduction


Nautilus extension to convert audio files to a different format using the GStreamer framework.


Compile it

The current released version of nautilus-sound-converter is 1.0.5, released July 6, 2010.

Download and extract the tarball, enter the program directory, then type:

$ ./configure $ make $ sudo make install (or run as root)

You can also fetch and compile the code from our git repository. To do so, type:

$ git clone git://git.gnome.org/nautilus-sound-converter

If compiling from a cloned git repository, simply replace the ./configure line with sh autogen.sh.

Dependencies

Nautilus-Sound-Converter depends on the following programs and libraries:

  • glib
  • GConf2
  • gstreamer
  • nautilus and nautilus-extensions
  • gnome-media
  • gstreamer-plugins-good
  • gstreamer-plugins-base
  • gstreamer-plugins-bad (optional)
  • gstreamer-plugins-ugly (optional)

Make sure you have installed the development packages for these libraries before compiling nautilus-sound-converter. You will also need the standard GNU and GNOME build tools installed.
