Most unprecedented application
Have an idea you think no one's ever had before? An innovative application that doesn't exist but that everyone needs? Something so cool it just has to be done?

This bounty will be awarded to the authors of the most unprecedented application. Submissions will be judged by a panel appointed by the GNOME board of directors.

Panel applet bundle system
A framework that lets developers implement panel extensions as simple to write single file bundles. Bundles will require no native code, probably written in Python (so they work between library versions etc), and will be user installable from web pages. Must include a tool for building the bundles.
Group mails by conversations in Evolution
Gmail provides an innovative new way of reading your emails: as conversations wherein the mails you send and the mails you receive in a given thread are all visible once, in chronological order. Implement this in Evolution.
Paste by example
Paste by example is a feature that allows users to learn "by example" the features of an office suite. It will let people learn to create any formatting or object seen in a document sent to them (or one they created 3 years ago). Office objects in the clipboard buffer (e.g. a graph or a formatted sentence) can be pasted into the current document, but step by step as a dynamically generated tutorial (showing the dialogues and commands users could have used to recreate the object in question). Feature can be implemented in OpenOffice.org or AbiWord.
Search party
Search party is a feature that allows adhoc chat rooms to form based on what people are searching for on the web (using e.g. Google). Implement the search party web browser extension for Firefox or Epiphany. Define the search party client/server "protocol" and implement a server.
Shared music spaces
Shared music spaces allow people in a shared chat room (e.g. an IM chat room) to listen to a music stream playing on any of their computers. Feature will be added using Rhythmbox and Gaim.
