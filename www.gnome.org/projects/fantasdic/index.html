Fantasdic

Dictionary application.

What is Fantasdic?

Fantasdic is a DICT client (RFC 2229). This is a dictionary application which uses the network to retrieve definitions. For example, nihongobenkyo.org is a server that provides Japanese dictionaries. Add it to the settings in Fantasdic and voil√†, you can now look up Japanese words using Fantasdic. Another well-known server with tons of dictionaries is dict.org.

Fantasdic is primarily targetting the GNOME desktop but it should work with other platforms, including Windows. Fantasdic is Free Software.

Features

Configure your dictionaries

Servers generally provide a lot of different dictionaries. With Fantasdic, you can select the dictionaries of your choice. You define a server (e.g. dict.org), select dictionaries of your choice and give a name to your dictionary.

You can this way define several dictionaries from several different servers. Dictionaries are of course not limited to language dictionaries.

Scan clipboard

Sometimes, reading through emails or web pages, you'll want to look up several words. When the "scan clipboard" option is enabled, select a word from any application, click "copy" and Fantasdic will look up this word for you.

Servers

Once Fantasdic is installed, you'll need to find servers. Some public servers with a wide range of dictionaries can be found on the internet. A list is available here. You may also run your own server on your local machine or network using the dictd server. Fantasdic is distributed with command-line tools that ease the conversion from existing dictionary formats (including stardict) to the format used by dictd.
