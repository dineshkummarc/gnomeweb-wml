Fantasdic

Dictionary application.

What is Fantasdic?

Fantasdic is a dictionary application primarily targetting the GNOME desktop but which should work with other platforms, including Windows. Fantasdic is Free Software.

Fantasdic connects to a DICT server to look up words in dictionaries provided by that server. So you might think of Fantasdic as a "dictionary browser".

Features

Make your own dictionaries

Servers generally provides a lot of different databases. With Fantasdic, you can create your customized dictionaries. You define a server (e.g. dict.org), select databases of your choice and give a name to your dictionary.

You can this way define several dictionaries from several different servers. Dictionaries are of course not limited to language dictionaries.

Scan clipboard

Sometimes, reading through emails or web pages, you will want to look up several words. When the "scan clipboard" option is enabled, select a word from any application, click "copy" and Fantasdic will look up this word for you.

Servers

Once Fantasdic is installed, you will want to find some servers. Some public servers with a wide range of dictionaries can be found on the internet. A list is available here. Or you may run your own server on your local machine or network using the dictd server. Fantasdic is distributed with command-line tools that ease the conversion from existing dictionary formats (including stardict) to the format used by dictd.
