Tracker is under an hard development that's why we are going to cover only some of the configurations.

Choose what to index!!!

You can configure the behavior of tracker using the configuration file : ~/.Tracker/tracker.cfg.

List all the directory that you want to index in the WatchDirectoryRoots field separated by a semicolon.

Default is set to your home directory.

WatchDirectoryRoots=/path/to/directory_1;/path/to/directory_2

List of directory roots to not index and not watch seperated by semicolons in the NoWatchDirectory

Default is not set

NoWatchDirectory=/path/to/directory_to_NOT_index_1;/path/to/directory_to_NOT_index_2
