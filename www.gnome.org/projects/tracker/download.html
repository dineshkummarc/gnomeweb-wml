Download

Here is the latest stable release for Tracker. Note that the deb's and the rpm's may be outdated.

  • Tarball : http://www.gnome.org/~jamiemcc/tracker/tracker-0.5.4.tar.gz

  • Debian : experimental debian repo can be found at : "deb http://debs.michaelbiebl.de/ unstable main"
  • Ubuntu :
    • Feisty : Tracker is in the universe.
      apt-get install tracker to install the indexer
      apt-get install tracker-utils to install some command-line tool
      apt-get install tracker-search-tool to install the GUI.
    • Edgy: repo "deb http://debs.michaelbiebl.de/ edgy main"
    • Dappper: repo "deb http://debs.michaelbiebl.de/ dapper main"
    If your architeture is !i386 you can use "deb-src http://debs.michaelbiebl.de/ dapper main"
    and recompiling yourself using "apt-get build-dep tracker && apt-get source -b tracker"
  • RPM (FC5/FC6): Can now be found in Fedora Extras
  • Gentoo: Tracker is in portage just emerge tracker to install it
