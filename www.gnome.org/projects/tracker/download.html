Download

Here is the latest stable release for Tracker.

Note that the deb's and the rpm's may be outdated.

Tarball

http://www.gnome.org/~jamiemcc/tracker/tracker-0.5.4.tar.gz

Debian i386

Experimental debian repo can be found at:
deb http://debs.michaelbiebl.de/ unstable main

Ubuntu i386

  • Feisty 7.04: Tracker is in the universe.

  • Edgy 6.10: repo

    deb http://debs.michaelbiebl.de/ edgy main

  • Dappper 6.06: repo

    deb http://debs.michaelbiebl.de/ dapper main

These packages have been digitally signed with this GPG key. Download the key and import it into apt using:

sudo apt-key add biebl.asc

Debian Based Distro or non i386 Debian and Ubuntu

Add the following to your /etc/apt/sources.list:

deb-src http://debs.michaelbiebl.de/ dapper main

Download the build-dependancies

sudo apt-get build-dep tracker

Compile the software:

sudo apt-get source -b tracker

Fedora 5 or 6

Install from Fedora Extras

Gentoo

Tracker is now included in portage. To install, run:

emerge tracker
