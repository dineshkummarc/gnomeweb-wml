What is it?

Tracker is a first class object database, extensible tag/metadata database, search tool and indexer.

It can trawl through your hard drive and index existing files and data stores.

Is Tracker able to detect the different types of files?

Short: Yes.

Long: Tracker is able to detect the mime type of the file. In other words this means that if you want to search only for "foobar" in your mp3 collection, you can do it. Tracker will show you only the your mp3's files, and not all the other files (Documents/Video/...) that have foobar as title or content [see the next one].

Tracker can index only the title or also the content of a file?

It depends on the type of the file. For text file tracker retrieve all the information that it can, from the title to the content of the file. For a music file, i.e.: mp3, it can retrieve the Artist, the Title of the song, the Album and more over.

Is there a Gui?

There is a Gtk2 gui for tracker. The name is tracker-search-tool. Check the screenshots to have an idea A Gui for the qt library is under development.
