Development

Retrieve current development code with the following command:

svn checkout http://svn.gnome.org/svn/tracker/trunk

cd into the root directory of the resulting sources and run:

./autogen.sh --prefix=/usr --sysconfdir=/etc

(See the README file for additional info)

ViewCVS web interface for svn: http://svn.gnome.org/viewcvs/tracker/

Some ideas about the new Features that want to be added to Tracker can be found on the wiki.
