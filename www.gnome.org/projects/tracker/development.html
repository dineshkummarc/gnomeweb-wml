Development

Currently Tracker 0.5.4 is in development with work ongoing to bring in new services like emails.

Hacking

The development code is available from the gnome svn, to check out:

svn checkout http://svn.gnome.org/svn/tracker/trunk

BUGS!!

To report a bug you can use:

  • the bugzilla on gnome http://bugzilla.gnome.org/browse.cgi?product=tracker
  • the Tracker Mailing list Tracker mailing list

Communication

You can contact us:

  • Using the Mailing List Tracker mailing list
  • On IRC: GimpNet, channel #tracker
