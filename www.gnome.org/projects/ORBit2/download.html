Download

ORBit2 depends on two further packages, glib2 and linc. The latter encapsulates basic connection management. Two common GNU/Linux distributions offer precompiled packages of ORBit2, additional developer packages are needed for application development only .

SuSE-8.1 (i386)

glib2 2.0.7 (devel) / linc 0.5.2 (devel) / orbit2 2.4.1 (devel)

Redhat-9.0 (i386)

glib2-2.2.1-1 (devel) / linc-1.0.1-1 (devel) / ORBit2-2.6.0-2 (devel)

Latest Stable Release

glib-2.2.0 / linc-1.1.0 / ORBit2-2.7.0

Joining Development

The shared code resides in GNOME CVS repository (CVS howto). To check out the latest version:

CVSROOT=':pserver:anonymous@anoncvs.gnome.org:/cvs/gnome'

cvs -d ${CVSROOT} login
(hit return as the password)
cvs -z3 -d ${CVSROOT} co glib
cvs -z3 -d ${CVSROOT} co linc
cvs -z3 -d ${CVSROOT} co ORBit2
