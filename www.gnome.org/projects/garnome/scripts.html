GARNOME Scripts
GARNOME Dependency Script by Boris Goldowsky (boris at alum mit edu).
Checks to see if you have all of the dependencies installed for GARNOME on a RPM based system.
GARNOME ChangeLog Script by Aschwin Van Der Woude (aschwin van der woude at linux-aktivaattori org).
Checks the version number of the packages in two different versions of GARNOME and outputs a Changelog.
GARNOME Build Count Script by Karsten (guenther) Bräckelmann (guenther at rudersport de).
Outputs the number of packages that have been built in GARNOME.
Fedora Dependency Install Scripts
GARNOME FC4 Dependency Install Script by Bob Kashani (bobk at ocf berkeley edu).
Installs all of the dependencies for GARNOME (desktop & platform) via YUM for Fedora Core 4.
Ubuntu Dependency Install Scripts
GARNOME Ubuntu 5 Dependency Install Script by Bob Kashani (bobk at ocf berkeley edu).
Installs all of the dependencies for GARNOME (desktop & platform) via APT-GET for Ubuntu 5.
