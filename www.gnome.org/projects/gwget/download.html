CVS Access
Gwget it's on Gnome CVS, module gwget.
To download it:

# export CVSROOT=:pserver:anonymous@anoncvs.gnome.org:/cvs/gnome
# cvs login
# cvs co gwget
# cd gwget
# ./autogen.sh
# ./configure
# make
# make install
