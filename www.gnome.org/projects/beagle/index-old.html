Quickly find the stuff you care about

Beagle is a search tool that ransacks your personal information space to find whatever you're looking for. Beagle can search in many different domains:


Find everything fast

Using Beagle, you can easily find:

  • documents
  • emails
  • web history
  • IM/IRC conversations
  • source code
  • images
  • music files
  • applications
  • ...and much more

Running Beagle

The most recent version of Beagle is 0.0.9, which was released on April 7, 2005. You can download the 0.0.9 tarball here.

The Beagle project wiki outlines where to get and install the software requirements. The wiki is full of useful information for people who want to compile Beagle themselves.

Older versions of Beagle are also available.

See Beagle Run

Nat Friedman has made a series of short movies that show Beagle in action. (Flash player required.)

Discussing Beagle

Discussions relating to Beagle take place in #dashboard on GIMPnet and on the dashboard-hackers mailing list.

Robert Love has set up Planet Beagle, an aggregator for the blogs various Beagle hackers.

There is a volunteer-maintained wiki devoted to Beagle and Dashboard development. If you are having problems, check there first.

Reporting bugs

If you find a bug in Beagle, please report it at the Beagle section of GNOME Bugzilla.

Hacking

Beagle is written in C# using Mono and Gtk#. The indexing is handled by DotLucene, a C# port of the Lucene indexer. The source code lives in the GNOME SVN repository in the module beagle.

Beagle is still in an early stage of development, and contributions are encouraged. Patches can be sent to the dashboard-hackers mailing list.
