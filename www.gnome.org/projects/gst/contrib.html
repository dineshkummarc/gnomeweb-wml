Getting involved with GST

There are several fields in which your help could be needed:

  • Testing: Testing that the tools work correctly on your system and either filing detailed bug reports or confirming that they work correctly is greatly appreciated. If you test a tool in an unsupported distro/system and find out that it works correctly, let us know so that we can update the supported tools matrix.
  • Porting:The backends are designed as to minimize the effort needed to support more distros/systems. A great way to contribute is to port the tools to another distribution/system.
  • Adding descriptions to the system services: The services tool requires short and long descriptions of every service for being really cool, if you see that the tool doesn't describe a service you have installed, then it's calling you to do it. you can easily help sending those descriptions to the project maintainer.
