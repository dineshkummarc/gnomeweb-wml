Getting involved with GST

There are several fields in which your help could be needed:

  • Testing: Testing that the tools work correctly on your system and either filing detailed bug reports or confirming that they work correctly is greatly appreciated. If you test a tool in an unsupported distro/system and find out that it works correctly, let us know so that we can update the supported tools matrix. You can see the latest version of the supported tools matrix in the release notes.
  • Porting:The backends are designed as to minimize the effort needed to support more distros/systems. A great way to contribute is to port the tools to another distribution/system.
  • Coding: The current status of the project requires more developers to carry out with the tools improvement. Helping to maintain or improve a tool would be an apreciated help.
