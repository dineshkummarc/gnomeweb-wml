Steps followed by the GST

  • The backend extracts the information and stores it in a hash table

  • The hash table is translated to XML and sent to the frontend

  • The frontend interprets the XML and shows the data to the user

  • When the user changes anything, these changes affect the xml

  • When the user hits 'apply', the changed XML is sent back to the backend

  • The backend translates the XML to his hash equivalent

  • The backend applies the changes to the system files
