GNOME Network Bug Report

You can help us reporting bugs in bugzilla

Firstly, please make sure that you have the latest CVS version of GNOME Network before reporting bugs.

The standard procedure for bug reporting is detailed at bugzilla.gnome.org please ensure that your bug hasn't already been logged by checking the list. It would also help if you CC. your mail to gnome-network-list@gnome.org.
