Stable release

The current stable release of GtkSourceView is 2.0.2

GtkSourceView 2.0 requires GTK+-2.8.x and libxml2 2.5.x.

You can download it from the GNOME FTP server.

Unstable release

The current unstable release of GtkSourceView is 2.1.2

GtkSourceView 2.1 requires GTK+-2.12.x and libxml2 2.5.x.

You can download it from the GNOME FTP server or check it out from the Subversion repository as shown below:

svn checkout http://svn.gnome.org/svn/gtksourceview/trunk gtksourceview

For more information about Subversion, see the instructions on how to use the repository

Older versions

Some applications still require GtkSourceView 1.x, an older stable version of GtkSourceView. You can have the run-time and development environments for both GtkSourceView 2.x and GtkSourceView 1.x installed simultaneously on your computer.

The latest release of the 1.x series is 1.8.3

GtkSourceView 1.8 requires GTK+-2.8.x, libxml2 2.5.x and libgnomeprint 2.8.x.

You can download it from the GNOME FTP server.

Binaries

GtkSourceView binaries are available on:

GNU/Linux and Unix
GtkSourceView is included in all the major flavors of GNU/Linux and Unix. Ask your vendor for more information about how to install GtkSourceView.
Windows
Binaries for Windows can be found on the GNOME FTP server (only a few versions are available).
