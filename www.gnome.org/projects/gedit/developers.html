Developers informations

Maintainer

The current maintainer of gedit is Paolo Maggi <paolo (at) gnome.org>.

Mailing list and IRC

A mailing list [Archives] is available to discuss developement of gedit itself, of plugins and for any question which regards gedit and its usage.
You may also join the #gedit channel on irc.gnome.org and have a chat with us.

Wiki

gedit has a wiki page on the official GNOME wiki, where you can find useful information about current maintainance work and future developement plans.

SVN access

gedit svn repository is hosted on svn.gnome.org.

You can get the code from GNOME svn with the following command:

svn co http://svn.gnome.org/svn/gedit/trunk gedit

Then to build enter the gedit directory and run the usual:

./autogen
make
make install

You can also browse the sources on-line using GNOME's ViewCVS service.

Bugs and Feature Requests

gedit bugs are tracked in GNOME's bugzilla. Here is a list of currently open issues.

Patches

You can submit patches to the gedit mailing list or in bugzilla.


Downloads

The latest release can always be found at the GNOME ftp site.

The latest stable release is 2.18.0 (release notes).

Feedback

Bugs should be reported to the GNOME Bug Tracking System.

You can get in touch with developers and other users by sending a mail to the gedit mailing list [Archives] or by joining the #gedit channel on irc.gnome.org.
