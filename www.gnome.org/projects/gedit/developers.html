CVS access

gedit cvs repository is hosted on cvs.gnome.org.

You can get the code from GNOME cvs with the following commands:

cvs -d:pserver:anonymous@anoncvs.gnome.org:/cvs/gnome login [just press enter when asked for a password]
cvs -z3 -d:pserver:anonymous@anoncvs.gnome.org:/cvs/gnome co gedit

Then to build enter the gedit directory and run the usual:

./autogen
make
make install

You can also browse the sources online using GNOME's ViewCVS service
