gedit's features

While aiming at simplicity and ease of use, gedit is a powerful general purpose text editor.
Currently it features:

  • Full support for internationalized text (UTF-8)
  • Syntax Highlighting
  • Undo/Redo, with configurable levels
  • File Revert
  • Clipboard (Cut/Copy/Paste)
  • Print and Print Preview support
  • Search and Replace
  • A complete preferences interface

Beside, gedit has a flexible plugin system which allows to extend the editor functionality. Some of the plugins that are packaged and installed with gedit include, among others:

  • Spell checker
  • Insert Date/Time
  • Word count
  • Change case of selected text
  • Indent/Unindent chuncks of text
  • "Diff" compares two documents or files
  • "Shell Output" inserts output from a shell command
  • "Tag List" inserts "tags" for many common markup languages (HTML, Latex, etc.)
