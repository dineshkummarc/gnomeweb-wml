Joraike comentaba en esta historia de deporteyciencia.com lo siguiente:
  • Para los taladros se puede hacer con una broca de 3'5. y se puede marcar con un clavo acerado, para que no se nos mueva la broca del taladro.
  • Tener cuidado con la remachadora, que se quede bien cogido el cable con el remache. si se os queda suelto el martillo es fundamental un par de golpes contra una superficie dura y el alumnio de los remaches se ajusta al conjunto varilla.
  • Yo no le he puesto plástico para recubrirla. He usado la misma lona que se utiliza para las colchonetas de los gimnasios o para el salto de altura en los centros escolares. No es muy gruesa la que yo he puesto pero tiene una resistencia mayor que el plástico. hay de mas grosos lo que no se es la referencias a la hora de pedirla en un almacén.

Licencia

ChronoJump es software libre distribuido bajo la Licencia GPL. El código fuente del programa y del cronómetro puede ser consultado, así como modificado y redistribuido.

Se facilitan instrucciones de construcción de la plataforma de contactos y el cronómetro.

No es posible verificar la validez y fiabilidad de un instrumento informático de medida si no se facilita el código fuente. Por esta razón ChronoJump es óptimo para la investigación.
