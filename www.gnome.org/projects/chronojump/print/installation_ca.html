Instal·lació

Ja podeu provar ChronoJump 0.3 a qualsevol PC.

GNU/Linux
  1. Instal·lar Mono.
  2. Descarregar: chronojump-monolive-03.tgz
Windows
  • Consultar aquest correu (en castellà): Chronojump-mono-live-cd 0.3 . Versión deChronojump fácil de usar para usuarios Windows

CVS
  • Si us considereu un usuari avançat, podeu provar el programa des d'un sistema operatiu GNU/Linux, instal·lant Mono, baixant-vos les fonts del CVS de Chronojump a GNOME, i executar ./chronojump.exe o compilar la pròpia versió make clean, make.

Podeu obtenir ajuda apuntant-vos a les llistes de correu.

Si desitgeu conèixer com avança el projecte, podeu consultar l'arxiu changelog (fent clic en el número que hi ha a la dreta de "Revisió").
