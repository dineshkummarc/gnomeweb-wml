Installation

Now you can try ChronoJump 0.4 from any PC.

GNU/Linux
  1. Install Mono and GTK#.
  2. Download Chronojump
Windows
  • Read this e-mail: Chronojump-mono-live-cd v.0.4

CVS
  • If you are an advanced user: from an operative system GNU/Linux, install Mono, download the Chronojump sources on the chronojump CVS on GNOME, and execute mono chronojump.exe or compile with make clean, make.

Help can be found subscribing to the mailing lists.

If you wish to know the advance of the project, please check on changelog file (click the number at the right hand corner of "Revision").
