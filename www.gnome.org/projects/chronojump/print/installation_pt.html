Instalação

Agora você pode provar o Chronojump 0.3 desde qualquer PC.

GNU/Linux
  1. Instale Mono.
  2. Download: chronojump-monolive-03.tgz
Windows
  • Ler este e-mail (em espanhol): Chronojump-mono-live-cd 0.3 . Versión deChronojump fácil de usar para usuarios Windows

CVS
  • Se você se considera um usuário experiente pode provar o programa com um sistema operacional GNU/Linux, instalando Mono, baixando as fontes do CVS de chronojump do GNOME, e executar ./chronojump.exe ou compilar sua própria versão com make clean, make.

Também se pode obter ajuda registrando-se nas listas de e-mail.

Se deseja conhecer a evolução do projeto, consulte o arquivo changelog (claque no número da direita de "Revisão").
