Si usted se considera un usuario avanzado, puede probar el programa desde un sistema operativo GNU/Linux, instalando Mono, bajándose las fuentes del SVN de chronojump en GNOME haciendo: svn checkout http://svn.gnome.org/svn/chronojump/trunk chronojump.

Puede obtener ayuda leyendo el manual, apuntándose a las listas de correo. Se recomienda leer las preguntas frecuentes.

Si desea conocer el avance del proyecto, puede consultar el archivo changelog.
