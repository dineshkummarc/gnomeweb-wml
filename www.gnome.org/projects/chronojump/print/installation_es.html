Instalación

Ya puede probar ChronoJump 0.5 en cualquier PC. Esta versión es totalmente usable. Consultad el listado de cambios respecto al la versión anterior.

Si usted se considera un usuario avanzado, puede probar el programa desde un sistema operativo GNU/Linux, instalando Mono, bajándose las fuentes del SVN de chronojump en GNOME, y ejecutar mono chronojump.exe o compilar su propia version con make clean, make.

Puede obtener ayuda leyendo el manual, apuntándose a las listas de correo. Se recomienda leer las preguntas frecuentes.

Si desea conocer el avance del proyecto, puede consultar el archivo changelog (clic en el número a la derecha de "Revision").
