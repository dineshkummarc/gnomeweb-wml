Instalación

Ya puede probar ChronoJump 0.3 en cualquier PC.

GNU/Linux
  1. Instalar Mono.
  2. Descargar: chronojump-monolive-03.tgz
Windows
  • Consultar este correo: Chronojump-mono-live-cd 0.3 . Versión deChronojump fácil de usar para usuarios Windows

CVS
  • Si usted se considera un usuario avanzado, puede probar el programa desde un sistema operativo GNU/Linux, instalando Mono, y bajándose las fuentes del CVS de chronojump:
    1. cvs -d:pserver:anonymous@cvs.software-libre.org:/cvsroot/chronojump login (cuando se le pida un password pulse 'enter').
    2. cvs -z3 -d:pserver:anonymous@cvs.software-libre.org:/cvsroot/chronojump co chronojump
    3. ejecutar ./chronojump.exe o compilar su propia version con make.

Puede obtener ayuda apuntándose a las listas de correo.

Si desea conocer el avance del proyecto, puede consultar el archivo changelog (clic en el número a la derecha de "Revision"), y los archivos de la lista chronojump-developers.
