Instalación

Versión imprimible
Ya puede probar ChronoJump 0.7 (13-Mayo-2008) en cualquier PC. Esta versión es totalmente usable. Consultad el listado de cambios respecto al la versión anterior.

Sistemas operativos
instalación
GNU/Linux Chronojump 0.7 Linux
Windows Chronojump 0.7 Windows
Maemo N770Consultar este correo: chronojump_mini for Maemo!! Chronojump_mini funcionando en Maemo!! (inglés y castellano)


Si usted se considera un usuario avanzado, puede probar el programa desde un sistema operativo GNU/Linux, instalando Mono, bajándose las fuentes del SVN de chronojump en GNOME haciendo: svn checkout http://svn.gnome.org/svn/chronojump/trunk chronojump.

Puede obtener ayuda leyendo el manual, apuntándose a las listas de correo. Se recomienda leer las preguntas frecuentes.

Si desea conocer el avance del proyecto, puede consultar el archivo changelog.

Premio


Chronojump gana premio de software libre 2007 en la categoría de educación.

Licencia

ChronoJump es software libre distribuido bajo la Licencia GPL. El código fuente del programa y del cronómetro puede ser consultado, así como modificado y redistribuido.

Se facilitan instrucciones de construcción de la plataforma de contactos y el cronómetro.

No es posible verificar la validez y fiabilidad de un instrumento informático de medida si no se facilita el código fuente. Por esta razón ChronoJump es óptimo para la investigación.
