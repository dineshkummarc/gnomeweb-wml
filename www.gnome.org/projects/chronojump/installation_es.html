Instalación

Versión imprimible
Ya puede probar ChronoJump 0.48 en cualquier PC. Esta versión es totalmente usable.

Sistemas operativos
Instrucciones
GNU/LinuxInstalar Mono y Gtk#, Descargar Chronojump v.048 para Linux
WindowsConsultar este correo: Chronojump 0.48 for windows released!! (inglés y castellano)

CVS
  • Si usted se considera un usuario avanzado, puede probar el programa desde un sistema operativo GNU/Linux, instalando Mono, bajándose las fuentes del CVS de chronojump en GNOME, y ejecutar mono chronojump.exe o compilar su propia version con make clean, make.

Puede obtener ayuda leyendo el manual, apuntándose a las listas de correo. Se recomienda leer las preguntas frecuentes.

Si desea conocer el avance del proyecto, puede consultar el archivo changelog (clic en el número a la derecha de "Revision").

Licencia

ChronoJump es software libre distribuido bajo la Licencia GPL. El código fuente del programa y del cronómetro puede ser consultado, así como modificado y redistribuido.

Se facilitan instrucciones de construcción de la plataforma de contactos y el cronómetro.

No es posible verificar la validez y fiabilidad de un instrumento informático de medida si no se facilita el código fuente. Por esta razón ChronoJump es óptimo para la investigación.
