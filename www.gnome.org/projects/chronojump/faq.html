Contact platform
  1. How can I build a contact platform?

    Follow the instruction on given on this web page: "Instruction on how to build a contact platform to measure jump capacity".

  2. Where can I buy a contact platform? Can I sell the one I built myself?

    We don´t trade the contact platforms, instead we advice those who want to use this method to build their proper platform. If any person or company ever decide to produce them by the method we have taught or any other compatible method and decide to commercialize, an aware can be given to us so that we can divulge this service even though we are not responsible for the quality of the products.

  3. Why does the steel has to be warmed?

    The steel of the contact platform will be submitted to a number of preassures, if warmed it will resiste better and it will also have a better capacity to return to its initial position.

  4. Why don´t you use photoelectric cells?

    Because so far no one has collaborated offering an information on how to build them, where to buy, prices, delay on detection of events…We are thankful to receive such collaboration.

  5. I have found a better manner to build the contact platform, where could I advertise it?

    Write an e-mail to the list chronojump-list and we will be trying to find a way to announce your proposal.

Cronometer Chronopic
  1. Why the need of a chronometer if any computer can cover this function?

    The modern operative systems are multifunction, this mean that the computer need a certain quantity of time to develop each function in a way that seems to be executing all functions at the same time. If the computer is in charge to receive the stimulus arrived from the contact platform, in certain occasions it may delay the events because it will be developing otherssubjects, this might mean problems of reliability due to the jumps of the same length of time could have different measurements on different circumstance.
    The output chronometer that we have built is special on receiving those events given on the contact platform, avoiding the reliability errors when registering the events.
    Moreover Chronopic permits the use of Chronojump or its reduced version: Chronojump-mini to be used in any electronic display, assuring a good compatibility toward the future.
    For more information, please read this article: "Proyecto Chronojump: Sistema de Medida de la Capacidad de Salto usando Software y Hardware Libres" (spanish).

  2. I have read that the current chronometer is a prototype, when will the final version be released?

    For the construction of chronopic it has been used a training card of robots Skypic, taking the chance of a free software, its construction has been documented and it is possible to acquire directly and it has been also validated.
    So far we have an expansion card that permits the Skypic connection with to contact platform and a software (firmware) that permits Skypic to be completely functional on the detection of stated and events on the contact platform.
    Soon a Chronopic card will be released using less instruction than Skypic and also reduced dimensions and it will be build easily or acquired by a reduced price (15-30 euros in Spain).
    For more information, please read the webpage: documentación de Chronopic 1.0 y el prototipo 1 (spanish).

  3. How can I build the chronometer?

    The complete information can be found in this webpage: documentación de Chronopic 1.0 y el prototipo 1 (spanish). If you do not have knowledge in electronics, ask for help on this subject to any body you know or a local company.

  4. Where can I buy the chronometer?

    The IFARA company from Madrid commercialize in Sapin the Skypic + card of expansion + Chronopic firmware (needed software for the card), cost of 70 euros. For more information: Adquisition of the Prototype I of chronopic (spanish).
    If you want to produce and sell Skypic + card of expansion + Chronopic firmware in Spain or any other country, please contact to us so that we can advertise the service.

Software Chronojump
  1. Can accede the source code and modify the program? And also sell it?

    Chronojump is a free software distributed under the terms of GPL license , and this permit you to study, modify, redistribute and sell according to the license above. For more information web de GPL on Wikipedia.

  2. ¿How much does it cost?

    On the web it will always have a free Chronojump version, but you can also commercialize if you wish a version, we recommend you to offer extra services to the clients that decide to buy them. You may offer courses to trainers, consultant, integration of the platform + chronometer + software on one machine, …you can also offer the program on any course, class or conference you are involved.

  3. In what operative systems and computer equipment does it work?

    Chronojump has been written for the programming platform MONO, that has been conceived in order to facilitate the work of the computer programs in different operative systems and computer equipments. So far it only works in the system GNU/Linux, it is found on a Live-CD that permits the use of Chronojump from any PC (within the operative systems like Linux, Windows, FreeBSD o cualquier otro).
    By spring 2006 we forecast the release of a new version for windows without the need of the Live-CD.

  4. How can I download the program?

    Accede the web page Chronojump install.

  5. Can I try Chronojump even though I don´t have neither the chronometer or the contact platform?

    Yes, it has been created a simulation mode of jumps and races of different lengths of time so that it is possible to try the program features. Moreover it incorporates a sample data base.

  6. When it will be the release of the 1.0 version and what benefits it will have?

    Nowadays (on the verge of releasing version 0.4) Chronojump already have more benefits than any other commercial or non – commercial resolutions that we know. But there are still a number of aspects to improve, as we can see on the following document roadmap.
    If you wish to ask the features please write us an e-mail to the mail list chronojump-list.

  7. How can I know the last news of the project?

    Subscribe or check the files on mailing lists. Or check changelog.

  8. Why is it called Chronojump it it also work to measure races?

    The initial idea was the creation of a software destined to measure temporal jump events, but it was seen that with a little more effort the same software can be used to chronometer all kinds of races. Even though we are now working on these bases soon it will be added the rhythm study, the initial efforts will be centered on jumps measurements, that is why Chronojump maintains its original name.

  9. I would like to collaborate, how can I do it?

    Write to the mail list chronojump-devel-list introduce yourself, explain why do you want to collaborate and what are your knowledge
