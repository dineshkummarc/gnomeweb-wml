Instalación

﻿Versión para imprimir
Xa pode probar ChronoJump 0.5 en calquera PC. Esta versión é totalmente funcional. Consultad el listado de cambios respecto al la versión anterior [translation pending].

Sistemas operativos
Instrucións
GNU/LinuxInstalar Mono, libgdiplus e Gtk#. Consulte este correo (inglés, castelán): Chronojump 0.5 released! Chronojump 0.5 liberado!
WindowsConsulte este correo: Chronojump 0.5 for Windows installation (inglés, castelán)


Se vostede se considera un usuario avanzado, pode probar o programa desde un sistema operativo GNU/Linux, instalando Mono, baixando as fontes do SVN de ChronoJump en GNOME, e executando mono chronojump.exe ou compilar a súa propia versión con make clean, make.

Pode obter axuda consultando o manual (castelán), apuntándose ás listas de correo. Recoméndase ler as preguntas frecuentes.

Se desexa coñecer o avance do proxecto, pode consultar o arquivo changelog (clic no número á dereita de "Revisión").

Licenza

ChronoJump é software libre distribuído baixo a Licenza GPL. O código fonte do programa e do cronómetro pódese tanto consultar como modificar e redistribuír.

Facilítanse instrucións para a construción da plataforma de contactos e do cronómetro.

Non é posible verificar a validez e fiabilidade dun instrumento informático de medida se non se facilita o código fonte. Por esta razón, ChronoJump é perfecto para a investigación.
