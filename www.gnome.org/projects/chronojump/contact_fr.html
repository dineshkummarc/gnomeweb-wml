Licence

ChronoJump est un logiciel libre distribué sous le permis de GPL. Le code source de programme et de chronomètre est disponible, il est également possible de modifier et redistribuer.

Des directives de construction accompagnent le chronomètre et la plateforme de contacts.

Il n'est pas possible de vérifier la validité et la fiabilité d'un instrument de mesure numérique si le code source n'est pas disponible. Par conséquent Chronojump est approprié aux investigations.
