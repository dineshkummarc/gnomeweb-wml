License

ChronoJump is free software  distributed under the GPL license. The program and chronometer source code is available, it is also possible to modify and redistribute.

The chronometer and the contacts platform come along with construction guidelines.

It is not possible to verify the validity and fiability of a digital measuring instrument if the source code is not available. Therefore Chronojump is appopriate to investigations.
