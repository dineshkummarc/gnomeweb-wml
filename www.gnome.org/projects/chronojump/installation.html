Installation

Printable version
Now you can try ChronoJump 0.3 from any PC.

GNU/Linux
  1. Install Mono.
  2. Download: chronojump-monolive-03.tgz
Windows
  • Read this e-mail (spanish): Chronojump-mono-live-cd 0.3 . Versión deChronojump fácil de usar para usuarios Windows

CVS
  • If you are an advanced user: from an operative system GNU/Linux, install Mono and download the Chronojump sources on the CVS:
    1. cvs - d:pserver:anonymous@cvs.software-libre.org:/cvsroot/chronojump login (when the password is requested press 'enter').
    2. cvs - z3 - d:pserver:anonymous@cvs.software-libre.org:/cvsroot/chronojump co chronojump
    3. Execute /chronojump.exe or compile with make.

Help can be found subscribing to the mailing lists.

If you wish to know the advance of the project, please check on changelog file (click the number at the right hand corner of "Revision"), and the chronojump developers mailing list archives.
