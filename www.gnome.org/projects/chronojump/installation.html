Installation

Printable version
Now you can try ChronoJump 0.52 (March-31-2007) from any PC. This version is fully usable. Read changes list here.

Operating systems
Installation instructions
GNU/LinuxInstall Mono, libgdiplus, and Gtk#. Read this e-mail: Chronojump 0.52 for Linux (New!) Installation
WindowsRead this e-mail: Chronojump 0.52 for Windows (New!) Installation


If you are an advanced user: from an operative system GNU/Linux, install Mono, download the Chronojump sources on the chronojump SVN on GNOME, and execute mono chronojump.exe or compile with make clean, make.

Help can be found reading the manual (spanish), subscribing to the mailing lists. We recommend you to read the FAQ.

If you wish to know the advance of the project, please check on changelog file (click the number at the right hand corner of "Revision").

License

ChronoJump is free software  distributed under the GPL license. The program and chronometer source code is available, it is also possible to modify and redistribute.

The chronometer and the contacts platform come along with construction guidelines.

It is not possible to verify the validity and fiability of a digital measuring instrument if the source code is not available. Therefore Chronojump is appopriate to investigations.
