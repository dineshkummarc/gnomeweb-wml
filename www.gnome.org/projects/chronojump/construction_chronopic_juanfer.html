Construction of Chronopic_juanfer prototype

Printable version
Juan Fernando Pardo has started to release an own Chronopic version, simpler and cheaper. This version is NOT YET VALIDATED.

He managed to build it in one hour, and the total price in Argentina is like 22 US$

If you want to use a validated Chronopic, but more expensive (70 Euros in Spain), check the construction page.

  • Los componentes están montados sobre una tarjeta "experimental"
  • Las líneas verdes, marrones y naranja son cables ubicados en la parte inferior de la placa soldados únicamente en los extremos

Lista de componentes: (pending to translate)
  • C1: Condensador multicapa de 100n
  • C2: Condensador electrolitico 1uF 63v
  • C3: Condensador electrolitico 1uF 50v
  • C4, C5, C6, C7: Condensador electrolitico 22uF 25v
  • C8, C9: Condensador cerámico 22pF
  • CL1, CL2: Clema (bornera) para circuito impreso de 2 vías
  • R1: Resistencia 10K ohmios 5% ¼ W
  • R2: Resistencia 147 ohmios ¼ W
  • R3: Resistencia 330 ohmios 5% ¼ W
  • S1, S2: Pulsador de tacto para circuito impreso normal abierto
  • L1: Diodo LED 3mm color verde
  • MAX232: Conversor RS232 - TTL MAX232
  • PIC16F876A: Pic de 28 pines estrecho
  • Tarjeta experimental 8cm x 5cm aproximadamente

License

ChronoJump is free software  distributed under the GPL license. The program and chronometer source code is available, it is also possible to modify and redistribute.

The chronometer and the contacts platform come along with construction guidelines.

It is not possible to verify the validity and fiability of a digital measuring instrument if the source code is not available. Therefore Chronojump is appopriate to investigations.
