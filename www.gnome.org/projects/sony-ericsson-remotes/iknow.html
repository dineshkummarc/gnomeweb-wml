Introduction


See them in action

This page contains application profiles for use with the remote control application on Sony Ericsson phones.


Supported phones

The first phone to support the remote control application was the Sony Ericsson K700. Most of the K-series, W-series and Z-series phones with Bluetooth should support the remote control application.


The full list of supported phones is listed in the release notes for the Bluetooth Remote Control application. We don't keep a list here, as it's ever changing. Check your manual if in doubt.

Supported applications

Currently, the only supported applications are Evince, the document viewer, Rhythmbox, the music management application, and Totem, the movie player.

Application definition files

The definition files are available in 3 different sizes: 128x160, 176x220 240x320

You will need to select the resolution that corresponds to your phone display. Here's the screen sizes for a select number of phones:

  • 128x160 pixels: K320, K510, W300 and Z530
  • 176x220 pixels: K530, K550, K600, K610, K630, K700, K750, V640, V800, W550, W600, W610, W660, W700, W710, W800, W810, Z520, Z525, Z550, Z558, Z610, Z710 and Z800 series
  • 240x320 pixels: C702, C902, K660, K770, K790, K800, K810, K850, S500, S700, T650, W580, W760, W830, W850, W880, W890, W900, W910, W980, Z750 and Z770 series

After downloading the files on your PC, you can copy the files to your phone by right-clicking on the file, and selecting "Send to..." and select Bluetooth as the medium. If you have already used the remote control application with your computer, that's all you need to do.

If you haven't setup your computer for it yet, once the file has been copied, you will need to setup the phone as an input device. In the Bluetooth preferences, in the Services tab, select Input and make sure the service is enabled, and click Add..., and follow the on-screen instructions. Your computer can now be remote controlled!
