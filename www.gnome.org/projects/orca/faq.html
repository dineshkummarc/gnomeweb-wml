Downloads

Orca is currently under development and works best on the GNOME 2.14 platform. It is currently not a sanctioned component of the GNOME desktop, but we are currently planning on working to make it a part of GNOME 2.16. If you'd like to support us in this effort, please drop us a line and let us know.

The latest stable sources can always be found at the GNOME ftp site. The latest stable release is orca-0.2.2 (changelog).

The brave can always obtain the latest sources from the 'orca' module in the GNOME CVS source code repository


Feedback

Bugs and feature/enhancement requests (RFEs) should be reported to the GNOME Bug Tracking System.

Instructions for creating patches can be found in the GNOME introduction to CVS.

You can get in touch with developers and other users by sending a mail to the Orca mailing list [Archives] or by joining the #orca channel on irc.gnome.org.
