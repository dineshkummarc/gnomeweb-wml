Downloads

Orca is part of the GNOME 2.16 plaform, but the Orca releases also work well on the GNOME 2.14 platform.

The latest stable sources can always be found at the GNOME ftp site.

The brave can always obtain the latest sources from the 'orca' module in the GNOME CVS source code repository

Please refer to the Orca Wiki and Orca FAQ for up-to-date information on Orca.


Feedback

Bugs and feature/enhancement requests (RFEs) should be reported to the GNOME Bug Tracking System.

Instructions for creating patches can be found in the GNOME introduction to CVS.

You can get in touch with developers and other users by sending an e-mail to the Orca mailing list [Archives] or by joining the #orca channel on irc.gnome.org.
