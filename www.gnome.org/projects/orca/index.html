Introduction to Orca

Orca is a flexible, extensible, and powerful assistive technology for people with visual impairments. Using various combinations of speech synthesis, braille, and magnification, Orca helps provide access to applications and toolkits that support the AT-SPI (e.g., the GNOME desktop).


HELP! I can't run Orca!

Orca is currently available in source code form, so it's not quite ready for the faint of heart yet. In addition, it works best on 32-bit systems that have GNOME 2.14 installed. Based upon discussions on the community orca-list, we are trying to keep up-to-date instructions for obtaining, building, installing, configuring, and running Orca on various distributions. These instructions are still under development, but we plan to include instructions for at least:

  • Open Solaris
  • Ubuntu 5.10 and Ubuntu 6.06
  • Fedora Core 4 and Fedora Core 5

In addition, you may also want to check out the FAQ.

Downloads

Orca is currently under development and works best on the GNOME 2.14 platform. It is currently not a sanctioned component of the GNOME desktop, but we are currently planning on working to make it a part of GNOME 2.16. If you'd like to support us in this effort, please drop us a line and let us know.

The latest stable sources can always be found at the GNOME ftp site. The latest stable release is orca-0.2.2 (changelog).

The brave can always obtain the latest sources from the 'orca' module in the GNOME CVS source code repository


Feedback

Bugs and feature/enhancement requests (RFEs) should be reported to the GNOME Bug Tracking System.

Instructions for creating patches can be found in the GNOME introduction to CVS.

You can get in touch with developers and other users by sending an e-mail to the Orca mailing list [Archives] or by joining the #orca channel on irc.gnome.org.
