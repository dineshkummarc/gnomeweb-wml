Introduction to Orca

Orca is a flexible, extensible, and powerful assistive technology for people with visual impairments. Using various combinations of speech synthesis, braille, and magnification, Orca helps provide access to applications and toolkits that support the AT-SPI (e.g., the GNOME desktop).

Please refer to the Orca Wiki and Orca FAQ for up-to-date information on Orca.


How can I help?

We need help with:

  • Testing

  • Script Writing

Please visit the Wiki for more information on Orca, such as design documentation and source code documentation. If you are interested in contributing to Orca, please drop us a line.

Downloads

Orca is part of the GNOME 2.16 plaform, but the Orca releases also work OK on the GNOME 2.14 platform.

The latest stable sources can always be found at the GNOME ftp site.

The brave can always obtain the latest sources from the 'orca' module in the GNOME SVN source code repository

Please refer to the Orca Wiki and Orca FAQ for up-to-date information on Orca.


Feedback

Bugs and feature/enhancement requests (RFEs) should be reported to the GNOME Bug Tracking System.

Instructions for creating patches can be found in the GNOME introduction to SVN.

You can get in touch with developers and other users by sending an e-mail to the Orca mailing list [Archives] or by joining the #orca channel on irc.gnome.org.
