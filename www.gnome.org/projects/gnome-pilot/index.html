What is gnome-pilot ?
gnome-pilot is a package of utilities for the 3Com PalmPilot and is a part of the GNOME project.

The goal is a daemon both for the single pilot user, but also for eg. office use, where many pilots and many people are involved. And a complete set of office ready extras.

Among the features that the package will offer, are

  • CORBA based daemon
  • Support for multiple pilots/users and cradles
  • Management of users an pilots
  • A suite of conduits for GNOME applications
  • GNOME compliant utilities

Components and their features
capplet
A control-center applet for managing a single user installation.
conduits
  • backup conduit
    • backs up bases, optionally only altered bases.
    • optionally delete local bases if deleted on the pilot.
  • file conduit
    • can install files.
    • can restore a entire directory to the pilot.
  • memopad conduit
    • now being rewritten for Think interoperability.
    • A version that synchronizes to local files has been readded.
  • email conduit
    • Sync your email to/from the pilot.
  • Expense conduit
    • Copies your expense entries to CSV files
  • GnomeCalendar & GnomeCard conduits
    • These are packaged in the gnome-pim package.
  • MAL
    • A conduit that synchronizes with a MAL server
GMC interaction
Installing gnome-pilot lets you install files from within GMC (as well as command line)
gpilotd
  • Handles multiple pilots
  • handles multiple cradles
applet
This is a panel applet, that indicates synchronization by switching color, also supports installing files by drag'n'drop.
