28 February 2000
module gnome-pilot-conduits
A new cvs module (gnome-pilot-conduits) has been made. It will contain 3rd party conduits and currently contains the MAL conduit.

Amongst other news (yes, it's been a long time) ; a new libgpilotd (the clientlib) and libgpilotdcm (conduit management lib) have been made, and much progress on the gui parts (capplets & applet) has been done.

We aim at having a 0.2 ready for April Gnome, so expect 0.1.50 within a week or two.

If you wan't to see what we're hacking at, check the TODO file in cvs, has item and to some degree progress indications.

23 September 1999
0.1.20 and gnomecal conduit
0.1.20 is there, no RPMs since the really interesting things aren't in gnome-pilot.

Gnome-pilots new conduit architecture (by jrb) is merged into the HEAD branch, and in gnome-pim (also cvs), you can now find the early sketches of a gnomecal conduit using this architecture.

The conduit is a long way from finished, but it can transfer records both ways, so give it a test if you want (backup and xcopilot).
