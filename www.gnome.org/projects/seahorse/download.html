Requirements

Make sure you have the following installed before you try and install Seahorse. You may need to install 'devel' or 'dev' versions of these packages if your distro requires this.

  • GTK 2.6+
  • DBUS 0.35+
  • GPGME 1.x
  • GnuPG 1.2.x or 1.4.x
  • libgnomeui 2.x
  • libglade 2.x
  • GConf 2.x
  • GnomeVFS 2.x
  • OpenSSH
  • Gecko (from Firefox 1.5 or later)
  • Epiphany 2.14 (or later)

Optional Dependencies

These are not required for a basic install of Seahorse, but additional features will be enabled if these are present on your system. Again you may need to install 'devel' or 'dev' versions of these packages if your distro requires.

  • OpenSSH
  • OpenLDAP 2.x (client libraries)
  • Gedit 2.6+
  • libnautilus-extension 2.9+
  • gnome-keyring
  • libsoup 2.2
  • libpanelapplet
  • Avahi 0.6+
  • libnotify 0.3+
