About

F-Spot is an application designed to provide personal photo management to the GNOME desktop. Features include import, export, printing and advanced sorting of digital images.

Download

F-Spot is still in the early phases of development. Prerelease versions of the software are available from the Gnome Archives
Download the latest F-Spot source
The newest release (0.0.13) Fixes several bugs from the previous release.

Hacking

F-Spot is available as Open Source sofware and contributions are encouraged. F-Spot is written in C# using Mono and Gtk#. The source code lives in the GNOME CVS repository in the module f-spot. Currently the closest thing to a roadmap the project has is the TODO file in CVS.

Reporting Bugs

Bugs in F-Spot should be reported in GNOME Bugzilla under the module f-spot. For more information about reporting bugs see the Bugzilla HOWTO.

Mailing List

For discussion about F-Spot subscribe to the F-Spot mailing list or browse the online archives

IRC

Right now most discussion relating to F-Spot use and development takes place in #f-spot on irc.gnome.org.
