F-Spot

F-Spot is an application designed to provide personal photo management to the GNOME desktop. Plans include import, export, printing and advanced sorting of digital images.

Contributing

F-Spot is still in its infancy and contributions are encouraged. F-Spot is written in C# using Mono and Gtk#. The source code lives in the GNOME CVS repository in the module f-spot. Currently the closest thing to a roadmap the project has is the TODO file in CVS.

Reporting Bugs

Bugs in F-Spot should be reported in GNOME Bugzilla under the module f-spot. For more information about reporting bugs see the Bugzilla HOWTO.
