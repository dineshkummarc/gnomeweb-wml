  • 2003-02-19: Released version 0.16
    Major delay in the release of 0.16 was caused by a nearly complete reworking of the network layer of Straw. The new release should be stabler and more responsive.
    Please note that the new version adds more dependencies: you now need the ADNS library and the corresponding Python binding.
    New stuff:
    • Asynchronous networking instead of synchronous networking in a thread.
    • Show a broken image symbol if there was a problem fetching the image. While waiting for an image to get downloaded, a waiting symbol is displayed.
    • Fixed about box, now it can be displayed multiple times per session.
    • Added German translation.
    • Improved error reporting: no more exception spew to stdout/err, try to show all the errors in the feed view and show error icons in the feed list. Store errors with feed information.
    • Don't re-fetch blogroll/subscriptions every time the user asks for them, only after polling
    • Find works now for freshmeat information, too
    • Save and load the main window size
    • Workaround for the broken gettext in Python < 2.2.2(?)
    • Cleaned up the code
