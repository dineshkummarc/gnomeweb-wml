Help with the development of Straw is welcome.

If you want to contribute code, kindly look at the TODO list included in the sources (not that you can't hack on anything you want) and the bugs tagged as enhancements in the Straw bug tracking system. Make sure you let us know by posting in the mailing list to avoid duplication of effort.

Besides from coding, you can also help by submitting and triaging bugs, translating Straw in your native language, or help us write user documentation which we currently don't have yet. Writing good documentation is a good place to start if you're new to the open-source model of development.

Translations

If Straw is not yet translated into your language, please feel free to translate the strings. Translators are encouraged to cooperate with the GNOME Translation Project. Additional information can also be found at the Translation Project page. The README also contains hints and tips on how to use setup.py (distutils) to build and help you with the translations.

Development

To get the development version, just follow the steps in http://developer.gnome.org/tools/svn.html. The module is called straw.
