Minimum Requirements

Requires

  • Python 2.4,
  • PyGTK 2.6,
  • GNOME-Python 2.6,
  • GnomePythonExtras 2.10,
  • Berkeley DB 3 or 4 and their Python bindings,
  • ADNS library and the corresponding Python binding are highly recommended for smoother operation, but Straw will work without them.

If you are previously running version 0.21.2 or older, you also need mxDateTime to convert the straw database to the new format. Otherwise, you don't have to install it.

Other Feed Aggregators

There are other open-source feed aggregators out there. If Straw does not satisfy your needs, then you should e-mail us! Let us know what you want and we'll see what we can do about it. Anyway, here are your alternatives for the meantime:

  • Liferea and Blam for GNOME
  • Akregator for KDE
  • RSSOwl which uses Java Swing
  • Google Reader
  • Bloglines - Web-based
