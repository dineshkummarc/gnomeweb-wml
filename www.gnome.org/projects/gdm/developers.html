Developers information

Getting Involved

To get involved, contact maintainer Brian Cameron

More information about getting involved with other GNOME apps (and many tips that apply to GDM) can be found at http://live.gnome.org/GettingInvolved. For GDM internals refer to the GDM documentation.


Debugging

In most ways, GDM is a typical GNOME App; you should refer to http://developer.gnome.org/debugging for more information on debugging it.

Translating

Just as many other GNOME applications, GDM is translated by the GNOME translation team.


Drawing Icons

All icons drawn for GDM should match the GNOME Icon Style Guide


Documentation

If you want to write documentation for GDM, you should join the GNOME Documenation team.


Downloads

GDM should come bundled with your distribution.

The latest releases can always be found at the GNOME ftp site.

The latest stable release is 2.8.0.7 (changelog).

The latest stable release is 2.13.0.8 (changelog).


Feedback

Bugs should be reported to the GNOME Bug Tracking System.

You can get in touch with developers and other users by sending a mail to the GDM mailing list [Archives].
