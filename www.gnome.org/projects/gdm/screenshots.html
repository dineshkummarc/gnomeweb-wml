Downloads

GDM should come bundled with your distribution.

The latest releases can always be found at the GNOME ftp site.

The latest stable release is 2.14.2 (changelog).

The latest unstable release is 2.13.0.10 (changelog).


Feedback

Bugs should be reported to the GNOME Bug Tracking System.

You can get in touch with developers and other users by sending a mail to the GDM mailing list [Archives].
