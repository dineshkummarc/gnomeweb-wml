Preferences

Power preferences is a simple program that lets the authorized user change the actions associated with each action. Devices that are not present are not shown, e.g. the UPS options are not visible to the user, unless one is plugged in, similarly, battery options are hidden for desktop users.

Downloads

GNOME Power Manager should come bundled with most modern distributions.

The latest source releases can always be found at the SourceForge site.

Make sure to get latest release of HAL, the service that GNOME Power Manager depends on.


Feedback

Bugs should be reported to the GNOME Bug Tracking System.

Please read the FAQ and the bug reporting guidelines before creating a new bug.

You can get in touch with developers and other users by sending a mail to the Power Manager mailing list [Archives].
