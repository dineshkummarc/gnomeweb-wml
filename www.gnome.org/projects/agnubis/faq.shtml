FAQ

What is Agnubis?

Agnubis is a WYSIWYG presentation program for the GNOME desktop environment= . It is free software, licensed under the GPL.

History

The idea for Agnubis was the result of there being no viable alternative for a presentation program for the GNOME desktop environment. Agnubis is a component of the GNOME-Office application suite.

What are or will be its main features?

  • Agnubis' primary file format is XML-based.
  • We will support a wide variety of graphics formats.
  • Incorporate helpful assistants (sometimes called wizards or druids) similar to those found in other programs.
  • It will use a left panel view listing the slides in the presentation, by title or thumbnail, using our canvas widget.
  • Principle features and tools we hope to include:
    • Diagrams
    • Images
    • Organizational charts
    • EDtles
    • 3D titles and other artistic writing.
    • Tables
    • Graphs
    • Videos, sounds, and coordinated multimedia
    • Text and all related components
    • Animations

What language is it programmed in?

Mostly C, using GTK2 for the user interface, as well as perl for scripting certain processes (like exporting various file formats).
