Building Agnubis

To build Agnubis you need the GNOME 2 development plattform installed. In addition to this Agnubis also depends on DiaCanvas2i and gdome2. We currently have our own patches version for use with glib2. This version must be compiled with the --enable-glib-2 parameter passed to configure. You find our version here. We hope to get our fixes into the main version of gdome soon.

There are some special libraries required to build agnubis: (you cannot get this from anywhere else.)

  • gdome2-agnubis-fixed.tar.gz A custom version of Gdome2 that works with gtk+2 and gnome2. (compile with --enable-glib-2)

If you try to take gdome2 from somewhere else than above link, then it is likely that it links to old glib and would result to not-too-well working agnubis, as it would crash on gnome initialisation when two glib versions are linked to the same binary.

Currently Agnubis can be downloaded from GNOME Hispano CVS.

We have anonynomous CVS set up with the following CVSROOT
cvs -d:pserver:anoncvs@cvs.es.gnome.org:/home/cvs/gnome
The password is: anoncvs
