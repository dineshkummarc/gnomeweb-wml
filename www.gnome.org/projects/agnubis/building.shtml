Building Agnubis

To build Agnubis you need the GNOME 2 development plattform installed. In addition to this Agnubis also depends on DiaCanvas2.

Currently Agnubis can be downloaded from GNOME Hispano CVS.

We have anonynomous CVS set up with the following CVSROOT
cvs -d:pserver:anoncvs@cvs.es.gnome.org:/home/cvs/gnome
The password is: anoncvs

For developers with a CVS account on Hispano GNOME CVS server instructions to be found here

Agnubis CVS is also available through CVSWEB for browsing. You can look at the Agnubis code here.
