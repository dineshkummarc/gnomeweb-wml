Gnome Games – Usage Survey

Gnome Games is a collection of games for the GNOME desktop. We want the games in GNOME to continue to entertain our users, therefore we are now conducting a survey to find out how we can improve Gnome Games. We would therefore appreciate if you could answer this survey, which consists of a two questions: which new game would like to see included in Gnome Games, and which of the current games do you play?

The suggested games are Gnome Hearts, PyChess, Monster Masher, Multiplication Puzzle, glChess and Gnome Sudoku. For more information about each of the suggested new games, please see the Games Wiki.

For more information about each of the existing Gnome games, please see the main page about Gnome Games. The results of this survey will be used to let us know which games we should continue focusing on, and will be used in the plan for new games. For questions about the survey, please send an email to the Gnome Games mailing list.

Thank you for participating in this survey for improving Gnome Games! Now, please answer the following questions, and press the "Vote" button:

Download

The latest releases can be found at the GNOME ftp site.

Remember that even releases are stable (e.g. 2.14.x) and odd numbered releases are for testing (e.g. 2.15.x).

Testing releases contain cool new features, but the stable releases have a lot less bugs.

Feedback

Bugs should be reported to the GNOME Bug Tracking System

You can contact the developers by sending a mail to the Gnome Games mailing list.

Contributing

If you would like to contribute to Gnome Games please see the developers guidelines at the Gnome Games Wiki.
