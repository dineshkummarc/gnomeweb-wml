Nautilus File Manager

Nautilus is the official file manager for the GNOME desktop. It is designed to be primarly a file manager,
but there are support for web and file viewing too. These pages are meant to help you find information about Nautilus

    Nautilus and Scripting

    Nautilus supports a high degree of scripting for quickly adding extra functionality. Many users have already taken advantage of this to create lots of nice scripts. You find these scripts on the G-Scripts page on Sourceforge.
