Developing Evince

The AUTHORS file has the current people in charge of the Evince project maintenance and development.

Getting Started

If you'd like to start working on Evince see the Testing section for how to get the latest CVS sources of Evince.

Getting in Touch

Evince hackers use the mailing list and irc channel #evince on the server irc.gnome.org for keeping in touch and discussing issues regarding the project.

The TODO List

Evince has a TODO list that should be updated frequently describing the work that the developers would like to undertake and will gladly accept patches for.

Contact

You can contact developers sending a mail to the Evince mailing list. You do not need to be subscribed.

We are also often available to chat on IRC.
