Downloading Gnumeric

You can download the gnumeric sources from: ftp://ftp.gnome.org/pub/GNOME/stable/sources/gnumeric Binaries for For Debian are maintained by Vincent. Premade binaries can be found in most distributions and for a up to date version Ximian offers Gnumeric as part of their Ximian GNOME distribution. You can also usually ifnd RPMS from RPM find. Development version

As with any other software in GNOME, you can fetch the latest development version of Gnumeric from the GNOME Anonymous CVS and from the GNOME CVS from the module "gnumeric".

To compile Gnumeric you will need the gnome-libs package and the gnome-xml package. The gnome-libs package in turn requires the glib, gtk+ and imlib packages.

Development version

As with any other software in GNOME, you can fetch the latest development version of Gnumeric via Anonymous CVS. It is in the module "gnumeric".

To compile Gnumeric you will need several packages :

glib > = 1.2.7 glib 1.3 and 2.0 are not supported
gtk+ > = 1.2.7 gtk+ 1.3 and 2.0 are not supported
gnome-libs > = 1.0.57 gnome-libs 2.0 is not supported
xml-i18n-tools > = 0.8 only required to build, not while running)
libglade > = 0.16
gal > = 0.8
gnome-print > = 0.25 2.0 is not supported
gnome-xml > = 1.8.14 gnome-xml 2.x not supported yet, also known as libxml
libole2 > = 0.2.3 also known as libgnomeole2
