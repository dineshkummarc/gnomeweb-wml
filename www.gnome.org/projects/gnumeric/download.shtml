Downloading Gnumeric

You can download the stable Gnumeric source from: http://ftp.gnome.org/pub/GNOME/sources/gnumeric/1.2 Packages for Debian are available in unstable (stable and testing still use 1.0) Ximian is also a good source of up to date packages. You can also usually find RPMS from RPM find.

Gnumeric releases:

  • gnumeric-1.2.0 is the latest stable release

Requirements

You can run Gnumeric 1.2.x on any system with GNOME-2.0, 2.2, or 2.4 libraries installed.

To build from source you will need either development packages or source for several packages :

libgsf >= 1.8.2
libxml2 >= 2.4.12 also known as gnome-xml
glib >= 2.0.0
atk >= 1.0.0
pango >= 1.0.0
gtk+ >= 2.0.0
libgnome >= 2.0.0
libgnomeui >= 2.0.0
libonobo >= 2.0.0
libbonoboui >= 2.0.0
libgnomeprint >= 2.3.0
libgnomeprintui >= 2.3.0
libglade >= 2.0.0
intltools >= 0.27.2 Only required to build

Optionally in the stable version you can add:

GDA >= 1.0.0 GDA plugin, for accessing databases
PyGtk >= 2.0.0 Experimental python scripting
guile >= 1.6 Optional, guile for writing functions

Development version

As with any other software in GNOME, you can fetch the latest development version of Gnumeric from the GNOME Anonymous CVS and from the GNOME CVS from the module "gnumeric". The upcoming development series 1.3.x will be based on GNOME-2.6 libraries, there are details in gnumeric/README regarding build requirements.
