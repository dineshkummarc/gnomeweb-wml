Downloading Gnumeric

You can download the stable gnumeric source from: ftp://ftp.gnome.org/pub/GNOME/stable/sources/gnumeric Packages for Debian are available in testing and unstable (I would reccomend AGAINST using the ancient 0.47 package in stable). Ximian is also a good source of up to date packages. You can also usually find RPMS from RPM find. Packages of the development version are available at ftp://ftp.gnome.org/pub/GNOME/pre-gnome2/sources/gnumeric

Development version

As with any other software in GNOME, you can fetch the latest development version of Gnumeric from the GNOME Anonymous CVS and from the GNOME CVS from the module "gnumeric". The current development tree is based on GNOME2 libraries, there are details in gnumeric/README regarding build requirements.

Prereqs

To compile Gnumeric 1.0.x you will need several packages :

glib >= 1.2.7 glib 1.3 and 2.0 are not supported
gtk+ >= 1.2.7 gtk+ 1.3 and 2.0 are not supported
gnome-libs >= 1.0.57 gnome-libs 2.0 is not supported
xml-i18n-tools >= 0.9 Only required to build
libglade >= 0.16
gal >= 0.16
gnome-print >= 0.29 2.0 is not supported
gnome-xml >= 1.8.14 gnome-xml 2.x not supported yet, also known as libxml
libole2 >= 0.2.4 also known as libgnomeole2

Optionally:

bonobo >= 1.0.9 Requires gnome-libs >= 1.2.7
Guppi >= 0.40.0 corresponds to gnumeric 0.72-0.76
gb == 0.0.17 Gnome Basic for macros
gda >= 0.2.92 GDA plugin, for accessing databases
guile >= 1.5 Optional DEVELOPMENT version of guile for writing functions
Evolution >= beta3 Email a spreadsheet directly from gnumeric
