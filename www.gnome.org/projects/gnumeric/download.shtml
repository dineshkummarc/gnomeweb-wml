Downloading Gnumeric

You can download the gnumeric sources from: ftp://ftp.gnome.org/pub/GNOME/stable/sources/gnumeric Binaries for Debian are maintained by Vincent. Premade binaries can be found in most distributions and for an up to date version Ximian offers Gnumeric as part of their Ximian GNOME distribution. You can also usually find RPMS from RPM find. Development version

As with any other software in GNOME, you can fetch the latest development version of Gnumeric from the GNOME Anonymous CVS and from the GNOME CVS from the module "gnumeric".

Prereqs

To compile Gnumeric you will need several packages :

glib >= 1.2.7glib 1.3 and 2.0 are not supported
gtk+ >= 1.2.7gtk+ 1.3 and 2.0 are not supported
gnome-libs >= 1.0.57gnome-libs 2.0 is not supported
xml-i18n-tools >= 0.9Only required to build
libglade >= 0.16
gal >= 0.16
gnome-print >= 0.292.0 is not supported
gnome-xml >= 1.8.14gnome-xml 2.x not supported yet, also known as libxml
libole2 >= 0.2.4also known as libgnomeole2
