Features

  • Core capabilities
    • Functions
      • 98% of builtin MS Excel functions, plus many not in MS Excel
    • Computation
      • Inter sheet dependencies
      • User defined names (for ranges and functions)
      • Arrays, array functions
      • Iterative expressions
    • Misc
      • Frozen Panes
      • Graphs - Bar/Col, Line, Scatter.
      • Document metadata (properties)
      • Cell contents
      • Toolbar for common operations
      • Zoom
      • Multiple Views
      • Center Across Selection
      • Cell Merging
      • Content validation
      • Autofilter
    • Sheet objects
      • Lines, arrows, ellipses, squares, pictures
      • Buttons, checkbuttons, scrollbars, lists
      • Bonobo Objects - images, minesweeper, etc.
    • Printing
      • Cell contents
      • Style information
      • Per sheet printer setups
      • Print preview
      • Print options
      • Scaling
        • Page orientation
        • 6 margins + paper size
        • Headers + footers
        • Grid lines, comments, headings
        • Print order
    • Formats
      • All builtin Excel-like formats: number, currency, accounting, date, time, percentage, fraction, scientific, text, special
      • Compatible format description language
    • Styles
      • Borders - outer and inner, colors, diagonals, 13 line types (all excel types)
      • Colors - foreground, background, 25 pattern types (all excel and applix types)
      • Fonts - font name, size, style (bold/italic/underline/strikethrough), color
      • Alignment - Horizontal (general, left, right, center, fill, justify), Vertical (top, center, bottom, justify), Wrap text
      • Validation
    • Autofill
      • Autofill for various sequences is supported
    • Automatic guessing of input
      • Gnumeric will try to parse user entered input as a number with a format applied (so you can enter dates directly, and they will be stored as a number + a format to render it).
  • International
    • Gnumeric has been localized to various languages, and we welcome new contributions. It handles numeric formats properly for various countries and languages.
  • Import filters
    • XML (gnumeric native format)
    • Excel >= 2 including XP
    • Applix 4 & 5
    • OpenCalc 1.x XML
    • Lotus 1-2-3 .wk1 format
    • Plan Perfect
    • Quattro Pro wb1, wb2, wb3
    • XBase files
    • Configurable text formats
    • GNU oleo
    • DIF
    • Sylk
    • HTML (various)
    • CSV (comma separated values)
  • Export filters
    • Excel >= 5 including dual stream and XP
    • XML (gnumeric format - all data except embedded objects)
    • HTML/XHTML
    • LaTeX, Troff
    • PDF, EPS, DVI
    • Configurable text formats
