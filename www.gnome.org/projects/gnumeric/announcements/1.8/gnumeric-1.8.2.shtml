Gnumeric 1.8.2 aka "" is now available.

The Gnumeric Team is pleased to announce the availability of Gnumeric version 1.8.2. At the same time we have also released Goffice 0.6.2 and LibGsf 1.14.8 which are recommended, but not outright required, for Gnumeric 1.8.2.

This release is a bugfix release mostly fixing a large number of problems with corrupted files. We thank Sum1 for digging those up.
