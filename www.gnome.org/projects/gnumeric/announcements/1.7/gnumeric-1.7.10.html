Gnumeric 1.7.10 aka "TBD" is now available.

The Gnumeric team is proud to announce the release of Gnumeric 1.7.10, a development release. This release requires the concurrently released Goffice 0.4.0, see below.

This release mainly improves the GtkPrint support introduced in 1.7.9. Note: gnome-print is still linked in via goffice, but this will change.

  • Andreas
    • Reimplement scaling.
    • Save orientation.
    • Show paper preview.
    • Show unit selector in print dialog.
    • Add margin adjustments.
  • Jean
    • New Time Series Analysis plugin.
  • Jody
    • Fix inadvertent relocation of reference on other sheets. [Bug 433907].

Gnumeric 0.4.0 aka "TBD" is now available.

  • Jean
    • Add support for cubic splines.
    • Fix graph rendering on big endian machines. [Bug 432532].
