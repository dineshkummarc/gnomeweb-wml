Requirements

You can run Gnumeric 1.1.x on any system with GNOME-2.0, 2.2, or 2.4 libraries installed.

To build from source you will need either development packages or source for several packages :

libgsf >= 1.8.2
libxml2 >= 2.4.12 also known as gnome-xml
glib >= 2.0.0
atk >= 1.0.0
pango >= 1.0.0
gtk+ >= 2.0.0
libgnome >= 2.0.0
libgnomeui >= 2.0.0
libbonobo >= 2.0.0
libbonoboui >= 2.0.0
libgnomeprint >= 2.3.0
libgnomeprintui >= 2.0.0
libglade >= 2.0.0
intltools >= 0.27.2 Only required to build

Optionally in the stable version you can add:

gda >= 1.0.0 GDA plugin, for accessing databases
gda >= 2.0.0 Experimental python scripting
guile >= 1.6 Optional, guile for writing functions
