Synopsis

IMLOG2(inumber)

Description

IMLOG2 returns the logarithm of a complex number in base 2.

  • If inumber is not a valid complex number, IMLOG2 returns #VALUE! error.
  • This function is Excel compatible.

Examples

IMLOG2("3-j") equals 1.66096-0.46419j.
