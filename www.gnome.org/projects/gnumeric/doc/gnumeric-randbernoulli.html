RandBernoulli

Name

RandBernoulli -- 

Synopsis

RandBernoulli(p)

Description

RandBernoulli returns a Bernoulli distributed random number.

If @p < 0 or @p > 1 RandBernoulli returns #NUM! error.

Examples

RandBernoulli(0.5).

See also

RAND, RANDBETWEEN.
