TRIM

Name

TRIM —

Synopsis

TRIM(text)

Description

TRIM returns @text with only single spaces between words.

* This function is Excel compatible.

Examples

TRIM(" a bbb cc") equals "a bbb cc".

See also

CLEAN, MID, REPLACE, SUBSTITUTE.
