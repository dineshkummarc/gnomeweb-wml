AREAS

Name

AREAS —

Synopsis

AREAS(reference)

Description

AREAS returns the number of areas in @reference.

Examples

AREAS((A1,B2,C3)) equals 3.

See also

ADDRESS, INDEX, INDIRECT, OFFSET.
