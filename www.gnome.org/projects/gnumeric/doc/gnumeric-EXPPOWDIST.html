EXPPOWDIST

Name

EXPPOWDIST —

Synopsis

EXPPOWDIST(x,a,b)

Description

EXPPOWDIST returns the probability density p(x) at @x for Exponential Power distribution with scale parameter @a and exponent @b.

Examples

EXPPOWDIST(0.4,1,2).

See also

RANDEXPPOW.
