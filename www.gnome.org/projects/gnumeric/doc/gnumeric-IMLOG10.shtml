IMLOG10

Name

IMLOG10 —

Synopsis

IMLOG10(inumber)

Description

IMLOG10 returns the logarithm of a complex number in base 10.

  • If inumber is not a valid complex number, IMLOG10 returns #VALUE! error.
  • This function is Excel compatible.

Examples

IMLOG10("3-j") equals 0.5-0.13973j.
