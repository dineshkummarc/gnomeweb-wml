IMSQRT

Name

IMSQRT —

Synopsis

IMSQRT(inumber)

Description

IMSQRT returns the square root of a complex number.

  • If inumber is not a valid complex number, IMSQRT returns #VALUE! error.
  • This function is Excel compatible.

Examples

IMSQRT("1+j") equals 1.09868+0.4550899j.
