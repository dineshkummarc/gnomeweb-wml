CONCATENATE

Name

CONCATENATE —

Synopsis

CONCATENATE(string1[,string2...])

Description

CONCATENATE returns the string obtained by concatenation of the given strings.

* This function is Excel compatible.

Examples

CONCATENATE("aa","bb") equals "aabb".

See also

LEFT, MID, RIGHT.
