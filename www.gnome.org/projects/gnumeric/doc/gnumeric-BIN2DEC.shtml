BIN2DEC — 
BIN2DEC(x)

1. Description

BIN2DEC function converts a binary number in string or number to its decimal equivalent.

  • This function is Excel compatible.

2. Examples

BIN2DEC(101) equals 5.

3. See also

DEC2BIN, BIN2OCT, BIN2HEX.
