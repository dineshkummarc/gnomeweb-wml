SQRT

Name

SQRT -- 

Synopsis

SQRT(x)

Description

SQRT function returns the square root of @x. If @x is negative, SQRT returns #NUM! error.

This function is Excel compatible.

Examples

SQRT(2) equals 1.4142136.
