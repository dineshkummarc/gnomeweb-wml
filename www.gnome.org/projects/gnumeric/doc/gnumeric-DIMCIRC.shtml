DIMCIRC — 
DIMCIRC(traffic,gos)

1. Description

DIMCIRC returns a number of circuits required from a number of traffic loads with gos grade of service.

2. Examples

DIMCIRC(24,1%) returns 35.

3. See also

OFFCAP, OFFTRAF, PROBBLOCK.
