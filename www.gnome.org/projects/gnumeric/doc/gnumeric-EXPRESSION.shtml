EXPRESSION

Name

EXPRESSION —

Synopsis

EXPRESSION(cell)

Description

EXPRESSION returns expression in cell as a string, or empty if the cell is not an expression.

Examples

entering '=EXPRESSION(A3)' in A2 = empty (assuming there is nothing in A3).

entering '=EXPRESSION(A2)' in A1 = 'EXPRESSION(A3)'.
