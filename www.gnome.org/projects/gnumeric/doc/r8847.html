OCT2DEC

Name

OCT2DEC -- 

Synopsis

OCT2DEC(x)

Description

OCT2DEC function converts an octal number in a string or number to its decimal equivalent. This function is Excel compatible.

Examples

OCT2DEC("124") equals 84.

See also

DEC2OCT, OCT2BIN, OCT2HEX.
