IMSEC

Name

IMSEC —

Synopsis

IMSEC(inumber)

Description

IMSEC returns the complex secant of the complex number z (inumber), where

sec(z) = 1/cos(z).

  • If inumber is not a valid complex number, IMSEC returns #VALUE! error.

Examples

IMSEC("2-j") equals -0.413149-0.687527j.

See also

IMCSC, IMCOT.
