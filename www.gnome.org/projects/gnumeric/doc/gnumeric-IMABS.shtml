IMABS

Name

IMABS —

Synopsis

IMABS(inumber)

Description

IMABS returns the absolute value of a complex number.

  • If inumber is not a valid complex number, IMABS returns #VALUE! error.
  • This function is Excel compatible.

Examples

IMABS("2-j") equals 2.23606798.

See also

IMAGINARY, IMREAL.
