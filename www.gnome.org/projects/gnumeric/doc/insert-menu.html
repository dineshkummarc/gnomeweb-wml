3.5. The Insert Menu

The Insert menu is

Figure 3-7. The Insert menu.

The menu choices are grouped into the following groups:

Insert into workbook.

These menu items allow the user to insert objects to the workbook.

Insert to current cell.

The menu items insert objects into the current cell.

This menu contains:

  • Sheet —Insert a new Sheet into the current workbook, immediately after the current worksheet.

  • Rows

  • Columns

  • Cells...

  • Graph

  • Object

  • Name

  • Add / Modify comment...

  • Special
