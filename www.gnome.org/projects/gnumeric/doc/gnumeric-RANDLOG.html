RANDLOG

Name

RANDLOG —

Synopsis

RANDLOG(p)

Description

RANDLOG returns a logarithmic-distributed random number.

* If @p < 0 or @p > 1 RANDLOG returns #NUM! error.

Examples

RANDHYPERG(0.72).

See also
