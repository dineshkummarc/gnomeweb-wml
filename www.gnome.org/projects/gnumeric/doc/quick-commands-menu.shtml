Using Menu Commands

The menus provide the simplest way to get to all of the commands provided by Gnumeric. These menus work like those in any GNOME application: clicking on the menu name causes a menu to open, dragging the mouse cursor onto the menu and then releasing the mouse button (or clicking again) while the cursor is above a menu entry causes that command to be executed.
