MULTINOMIAL

Name

MULTINOMIAL —

Synopsis

MULTINOMIAL(value1, value2, ...)

Description

MULTINOMIAL returns the ratio of the factorial of a sum of values to the product of factorials.

* This function is Excel compatible.

Examples

MULTINOMIAL(2,3,4) equals 1260.
