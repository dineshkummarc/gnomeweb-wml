3.4. Using Commands

The commands provided by Gnumeric can be accessed using several methods. These methods are explained here. The most important commands are explained in the rest of this chapter. All of the commands are explained in later chapters of this manual.

3.4.1. Using Menu Commands

The menus provide the simplest way to get to all of the commands provided by Gnumeric. These menus work like for any GNOME application: clicking on the menu name causes it to open a list of menu entries, dragging down and releasing (or clicking again) on a menu entry causes that command to be exectuted. For further information, see Introduction to Menus.

3.4.2. Using Toolbar Button Commands

The buttons on the toolbars are quite simple to use. Simply pressing one of the buttons will perform the command, either directly or by opening a dialog window. For further information, see The Gnumeric Toolbars.

3.4.3. Using Context Menu Commands

In many situations, Gnumeric provides a menu right under the mouse cursor if the right hand mouse button is clicked. This menu contains different entries depending on the placement of the mouse cursor when the right hand mouse button is clicked. For further information, see Context Menus in Gnumeric.

3.4.4. Using Keyboard Shortcut Commands

Certain common commands can be triggered using a combination of keys. The menu entries are often followed by a combination of keys which can be used to trigger that command. For instance, saving the file which is currently being used can be performed by typing jointly the control key and the s key (i.e. Ctrl+S). For further information, see Gnumeric Keybindings.
