SQRT

Name

SQRT -- 

Synopsis

SQRT(x)

Description

SQRT function returns the square root of @x. This function is Excel compatible.

If @x is negative, SQRT returns #NUM! error.

Examples

SQRT(2) equals 1.4142136.
