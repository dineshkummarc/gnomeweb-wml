CUM_BIV_NORM_DIST

Name

CUM_BIV_NORM_DIST —

Synopsis

CUM_BIV_NORM_DIST(a,b,rho)

Description

CUM_BIV_NORM_DIST calculates the cumulative bivariatenormal distribution from parameters a, b & rhoThe return value is the probability that two random variables with correlation @rho are respectively each less than @a and@b
