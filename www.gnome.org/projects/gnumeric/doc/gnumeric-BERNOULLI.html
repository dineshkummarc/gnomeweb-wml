BERNOULLI

Name

BERNOULLI —

Synopsis

BERNOULLI(k,p)

Description

BERNOULLI returns the probability p(k) of obtaining @k from a Bernoulli distribution with probability parameter @p.

* If @k != 0 and @k != 1 BERNOULLI returns #NUM! error.

* If @p < 0 or @p > 1 BERNOULLI returns #NUM! error.

Examples

BERNOULLI(0,0.5).

See also

RANDBERNOULLI.
