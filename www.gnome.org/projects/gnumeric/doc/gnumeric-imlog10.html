IMLOG10

Name

IMLOG10 -- 

Synopsis

IMLOG10(inumber)

Description

IMLOG10 returns the logarithm of a complex number in base 10.

This function is Excel compatible.

Examples

IMLOG10("3-j") equals 0.5-0.13973j.

See also

IMLN, IMLOG2.
