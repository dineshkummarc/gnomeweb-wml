Chapter 2. Entering data.

2.1. Strings and numbers

The basic forms of data that can be entered into a cell include strings, numbers, formulas, and cell references.

The simplest of these types are strings and numbers. They are just constant values that are stored in a cell. There is no special syntax for this kind of data. The only limitation is that strings and numbers cannot contain newlines or begin with a '='. The '=' is used to denote cells that contain formulas or cell references.

See Figure 2-1 for an example of entering a string.

Figure 2-1. Entering strings and numbers.

2.1.1. Dates and Numbers

Dates and numbers are a special case of strings that are automatically converted to numerical types. If a string entered matches one of the formats for dates or times, the string is converted into a serial number (representative of the time in the string) and the cell format is set to that of the original string.

To make use of this feature, the user just needs to enter in a number in a recognizable format. Gnumeric will automatically convert it into an appropriate serial number. Because of this, it is possible to enter a date or time in a human readable format, and have it stored in a numerical format inside of Gnumeric.

The interesting part of this is that a user entered string, can be autofilled (see Chapter 4) properly.

See Example 5-3 for some examples of the many date and time formats that are accepted by Gnumeric.

2.1.2. Editing of cells.

Entering data in a cell

  1. Activate the cell by clicking on it with the white-cross cursor. This will make the cell become the active cell.

  2. Type in the data or formula.

See Section 2.2 for more detail on entering formulas into cells.

2.1.3. Editing commands

After selecting the cell and initially entering the data, the user is in a limited editing mode, designed to aid in entering data which may contain cell references. The main keys for editing mistakes in this mode is the backspace key.

2.1.3.1. Advanced Editing mode

For a more complete set of editing options, especially for longer entries, there is an advanced editing mode.

This mode is accessed by pressing F2. This shifts the focus from the current cell to the editing region in the worksheet. This allows more complete control over the text and data in the cell. This includes the use of the cursor keys to position the cursor within the data in the cell. See Section 1.4.1 For more info on the data entry area.

Backspace

Delete the character to the left of the cursor.

Cursor keys

Move the cursor appropriately.

Ctrl+K

Delete to the end of the line

TIP
 

All the normal key bindings for Gnome application entry boxes also apply here.

2.1.3.2. Error Messages

If the user enters a formula or set of data that is not acceptable, Gnumeric will display an error message in the cell.
