SIN

Name

SIN —

Synopsis

SIN(x)

Description

SIN function returns the sine of @x, where @x is given in radians.

* This function is Excel compatible.

Examples

SIN(0.5) equals 0.479426.

See also

COS, COSH, SINH, TAN, TANH, RADIANS, DEGREES.
