IMCSC

Name

IMCSC —

Synopsis

IMCSC(inumber)

Description

IMCSC returns the complex cosecant of the complex number z (inumber), where

csc(z) = 1/sin(z).

  • If inumber is not a valid complex number, IMCSC returns #VALUE! error.

Examples

IMCSC("2-j") equals 0.635494-0.221501j.

See also

IMSEC, IMCOT.
