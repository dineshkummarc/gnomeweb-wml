MOD

Name

MOD -- 

Synopsis

MOD(number,divisor)

Description

MOD function returns the remainder when @divisor is divided into @number.

This function is Excel compatible.

MOD returns #DIV/0! if divisor is zero.

Examples

MOD(23,7) equals 2.

See also

INT, FLOOR, CEIL.
