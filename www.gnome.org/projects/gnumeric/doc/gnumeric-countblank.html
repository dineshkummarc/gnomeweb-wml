COUNTBLANK

Name

COUNTBLANK -- 

Synopsis

COUNTBLANK(range)

Description

COUNTBLANK returns the number of blank cells in a @range.

This function is Excel compatible.

Examples

COUNTBLANK(A1:A20) returns the number of blank cell in A1:A20.
