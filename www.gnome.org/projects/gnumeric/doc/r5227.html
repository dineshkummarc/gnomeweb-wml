DAY

Name

DAY -- 

Synopsis

DAY (serial_number)

Description

Converts a serial number to a day.

Note that Gnumeric will perform regular string to serial number conversion for you, so you can enter a date as a string.

Examples

See also

MONTH, TIME, NOW, YEAR.
