Using Keyboard Shortcut Commands

Certain common commands can be triggered using a combination of keys. The menu entries are often followed by a combination of keys which can be used to trigger that command. For instance, saving the file which is currently being used can be performed by typing jointly the control key and the s key (i.e. Ctrl+S).
