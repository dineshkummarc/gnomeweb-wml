IMREAL

Name

IMREAL —

Synopsis

IMREAL(inumber)

Description

IMREAL returns the real part of a complex number.

  • If inumber is not a valid complex number, IMREAL returns #VALUE! error.
  • This function is Excel compatible.

Examples

imreal("132-j") equals 132.
