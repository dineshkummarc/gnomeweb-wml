QUOTIENT

Name

QUOTIENT -- 

Synopsis

QUOTIENT(num,den)

Description

QUOTIENT function returns the integer portion of a division. @num is the divided and @den is the divisor. This function is Excel compatible.

Examples

QUOTIENT(23,5) equals 4.
