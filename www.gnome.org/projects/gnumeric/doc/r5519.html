CEIL

Name

CEIL -- 

Synopsis

CEIL(x)

Description

CEIL function rounds @x up to the next nearest integer.

This function is Excel compatible.

Examples

CEIL(0.4) equals 1.

CEIL(-1.1) equals -1.

CEIL(-2.9) equals -2.

See also

ABS, FLOOR, INT.
