TIME

Name

TIME -- 

Synopsis

TIME (hours,minutes,seconds)

Description

TIME returns a fraction representing the time of day.

This function is Excel compatible.

Examples

TIME(3, 5, 23) equals 3:05AM.

See also

HOUR.
