BIN2DEC

Name

BIN2DEC -- 

Synopsis

BIN2DEC(x)

Description

BIN2DEC function converts a binary number in string or number to its decimal equivalent. This function is Excel compatible.

Examples

BIN2DEC(101) equals 5.

See also

DEC2BIN, BIN2OCT, BIN2HEX.
