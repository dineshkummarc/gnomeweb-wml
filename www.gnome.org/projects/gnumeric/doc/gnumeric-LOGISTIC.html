LOGISTIC

Name

LOGISTIC —

Synopsis

LOGISTIC(x,a)

Description

LOGISTIC returns the probability density p(x) at @x for a logistic distribution with scale parameter @a.

Examples

LOGISTIC(0.4,1).

See also

RANDLOGISTIC.
