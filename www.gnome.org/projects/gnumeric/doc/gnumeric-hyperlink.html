HYPERLINK

Name

HYPERLINK -- 

Synopsis

HYPERLINK(link_location, optional_label)

Description

HYPERLINK function currently returns its 2nd argument, or if that is omitted the 1st argument.

Examples

HYPERLINK("www.gnome.org","GNOME").
