IMAGINARY

Name

IMAGINARY -- 

Synopsis

IMAGINARY(inumber)

Description

IMAGINARY returns the imaginary coefficient of a complex number.

This function is Excel compatible.

Examples

IMAGINARY("132-j") equals -1.

See also

IMREAL.
