IMCOT

Name

IMCOT —

Synopsis

IMCOT(inumber)

Description

IMCOT returns the complex cotangent of the complex number z (@inumber), where

cot(z) = 1/tan(z).

* If @inumber is not a valid complex number, IMCOT returns #VALUE! error.

Examples

IMCOT("2-j") equals -0.171384+0.821330j.

See also

IMSEC, IMCSC.
