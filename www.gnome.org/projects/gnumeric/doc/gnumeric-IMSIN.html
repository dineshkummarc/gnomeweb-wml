IMSIN

Name

IMSIN —

Synopsis

IMSIN(inumber)

Description

IMSIN returns the sine of a complex number.

* If @inumber is not a valid complex number, IMSIN returns #VALUE! error.

* This function is Excel compatible.

Examples

IMSIN("1+j") equals 1.29846+0.63496j.

See also

IMCOS, IMTAN.
