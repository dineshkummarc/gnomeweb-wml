NORMSINV

Name

NORMSINV -- 

Synopsis

NORMSINV(p)

Description

NORMSINV function returns the inverse of the standard normal cumulative distribution. @p is the given probability corresponding to the normal distribution. This function is Excel compatible.

If @p < 0 or @p > 1 NORMSINV returns #NUM! error.

Examples

NORMSINV(0.2) equals -0.841621234.
