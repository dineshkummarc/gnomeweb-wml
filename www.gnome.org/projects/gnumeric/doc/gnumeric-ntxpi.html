NT_PI

Name

NT_PI -- 

Synopsis

NT_PI(n)

Description

The NT_PI function returns the number of primes less than or equal to @n.

This function only takes one argument.

See also

ITHPRIME, NT_PHI, NT_D, NT_SIGMA.
