IMPRODUCT

Name

IMPRODUCT -- 

Synopsis

IMPRODUCT(inumber1[,inumber2,...])

Description

IMPRODUCT returns the product of given complex numbers. This function is Excel compatible.

Examples

IMPRODUCT("2-j","4-2j") equals 6-8j.
