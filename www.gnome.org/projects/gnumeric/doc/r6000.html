INT

Name

INT -- 

Synopsis

INT(a)

Description

INT function rounds @a now to the nearest integer where `nearest' implies being closer to zero. INT is equivalent to FLOOR(a) for @a >= 0, and CEIL(a) for @a < 0. This function is Excel compatible.

Examples

INT(7.2) equals 7.

INT(-5.5) equals -6.

See also

FLOOR , CEIL , ABS
