FIND

Name

FIND -- 

Synopsis

FIND(string1,string2[,start])

Description

FIND returns position of @string1 in @string2 (case-sesitive), searching only from character @start onwards (assumed 1 if omitted).

Examples

FIND("ac","Jack") equals 2.

See also

EXACT, LEN, MID, SEARCH.
