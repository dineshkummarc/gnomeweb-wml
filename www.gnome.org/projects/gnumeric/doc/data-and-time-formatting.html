Date and Time formatting

Gnumeric supports a wide variety of date and time formatting.

These formats can be used to convert an existing time into a new more readable format, or dates can be entered in these formats and Gnumeric will automatically convert them to a serial number with the proper cell format. See the Section called Dates and Numbers in the chapter called Entering data for more information.

In Example 3 'd' is a day, 'm' is a month, 'y' is a year, 'h' is an hour, and 's' is a second. The default set of formats is locale specific. Gnumeric attempts to guess whether the locale puts months before days or vice versa.
