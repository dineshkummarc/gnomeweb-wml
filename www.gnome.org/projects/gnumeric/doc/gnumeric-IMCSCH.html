IMCSCH

Name

IMCSCH —

Synopsis

IMCSCH(inumber)

Description

IMCSCH returns the complex hyperbolic cosecant of the complex number z (@inumber), where

csch(z) = 1/sinh(z).

* If @inumber is not a valid complex number, IMCSCH returns #VALUE! error.

Examples

IMCSCH("1+j") equals 0.303931-0.621518j.

See also

IMSECH, IMCOTH.
