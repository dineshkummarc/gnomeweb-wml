RandBinom

Name

RandBinom -- 

Synopsis

RandBinom(p,trials)

Description

RandBinom returns a binomialy distributed random number.

If @p < 0 or @p > 1 RandBinom returns #NUM! error. If @trials < 0 RandBinom returns #NUM! error.

Examples

RandBinom(0.5,2).

See also

RAND, RANDBETWEEN.
