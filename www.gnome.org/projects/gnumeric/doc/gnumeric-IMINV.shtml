IMINV

Name

IMINV —

Synopsis

IMINV(inumber)

Description

IMINV returns the the inverse, or reciprocal, of the complex number z (inumber), where

1/z = (x - i y)/(x^2 + y^2).

  • If inumber is not a valid complex number, IMINV returns #VALUE! error.

Examples

IMINV("1-j") equals 0.5+0.5j.
