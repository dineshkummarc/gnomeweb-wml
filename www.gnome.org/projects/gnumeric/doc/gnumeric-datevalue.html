DATEVALUE

Name

DATEVALUE -- 

Synopsis

DATEVALUE(date_str)

Description

DATEVALUE returns the serial number of the date. @date_str is the string that contains the date.

This function is Excel compatible.

Examples

DATEVALUE("1/1/1999") equals 36160.
