FIB

Name

FIB —

Synopsis

FIB(number)

Description

FIB function computes Fibonacci numbers.

* If @number is not an integer, it is truncated.

* If @number is negative or zero FIB returns #NUM! error.

Examples

FIB(12) equals 144.
