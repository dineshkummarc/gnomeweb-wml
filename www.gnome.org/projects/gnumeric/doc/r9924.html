MODE

Name

MODE -- 

Synopsis

MODE(n1, n2, ...)

Description

MODE returns the most common number of the data set. If the data set has many most common numbers MODE returns the first one of them.

Strings and empty cells are simply ignored. If the data set does not contain any duplicates MODE returns #N/A! error. This function is Excel compatible.
