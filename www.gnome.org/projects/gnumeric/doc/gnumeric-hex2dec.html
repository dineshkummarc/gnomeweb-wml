HEX2DEC

Name

HEX2DEC -- 

Synopsis

HEX2DEC(x)

Description

HEX2DEC function converts a hexadecimal number to its decimal equivalent.

This function is Excel compatible.

Examples

HEX2DEC("2A") equals 42.

See also

DEC2HEX, HEX2BIN, HEX2OCT.
