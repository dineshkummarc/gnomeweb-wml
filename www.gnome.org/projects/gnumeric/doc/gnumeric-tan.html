TAN

Name

TAN -- 

Synopsis

TAN(x)

Description

TAN function returns the tangent of @x, where @x is given in radians.

This function is Excel compatible.

Examples

TAN(3) equals -0.1425465.

See also

TANH, COS, COSH, SIN, SINH, DEGREES, RADIANS.
