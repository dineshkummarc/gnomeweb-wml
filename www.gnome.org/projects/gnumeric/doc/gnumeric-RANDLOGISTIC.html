RANDLOGISTIC

Name

RANDLOGISTIC —

Synopsis

RANDLOGISTIC(a)

Description

RANDLOGISTIC returns a logistic-distributed random number. The distribution function is,

p(x) dx = { exp(-x/a) over a (1 + exp(-x/a))^2 } dx for -infty < x < +infty.

Examples

RANDLOGISTIC(1).

See also

RAND.
