Synopsis

BETALN(a,b)

Description

BETALN function returns the natural logarithm of the absolute value of the beta function.

  • If a, b, or (a + b) are non-positive integers, BETALN returns #NUM!

Examples

BETALN(2,3) equals -2.48.

BETALN(-0.5,0.5) equals #NUM!.
