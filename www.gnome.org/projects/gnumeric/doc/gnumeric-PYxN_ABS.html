PY_N_ABS

Name

PY_N_ABS —

Synopsis

PY_N_ABS (num)

Description

Return the absolute value of number.

This is a totally pointless function. But it illustrates how to invoke a gnumeric function from Python.

Examples

PY_N_ABS(-5) equals 5)
