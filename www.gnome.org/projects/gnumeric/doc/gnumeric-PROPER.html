PROPER

Name

PROPER —

Synopsis

PROPER(string)

Description

PROPER returns @string with initial of each word capitalised.

* This function is Excel compatible.

Examples

PROPER("j. f. kennedy") equals "J. F. Kennedy".

See also

LOWER, UPPER.
