IMCOTH

Name

IMCOTH —

Synopsis

IMCOTH(inumber)

Description

IMCOTH returns the complex hyperbolic cotangent of the complex number z (inumber) where,

coth(z) = 1/tanh(z).

  • If inumber is not a valid complex number, IMCOTH returns #VALUE! error.

Examples

IMCOTH("1+j") equals 0.868014-0.217622j.

See also

IMSECH, IMCSCH.
