PI

Name

PI -- 

Synopsis

PI()

Description

PI functions returns the value of pi.

This function is called with no arguments.

This function is Excel compatible, except that it returns pi with a better precision.

Examples

PI() equals 3.141593.

See also

SQRTPI.
