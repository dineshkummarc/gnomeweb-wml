COLUMNS

Name

COLUMNS —

Synopsis

COLUMNS(reference)

Description

COLUMNS function returns the number of columns in area or array reference.

* If @reference is neither an array nor a reference nor a range, COLUMNS returns #VALUE! error.

Examples

COLUMNS(H2:J3) equals 3.

See also

COLUMN, ROW, ROWS.
