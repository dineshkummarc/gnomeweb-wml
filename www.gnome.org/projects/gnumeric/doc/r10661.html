OCT2HEX

Name

OCT2HEX -- 

Synopsis

OCT2HEX(number[,places])

Description

The OCT2HEX function converts an octal number to a hexadecimal number. @places is an optional field, specifying to zero pad to that number of spaces. This function is Excel compatible.

If @places is too small or negative #NUM! error is returned.

Examples

OCT2HEX(132) equals 5A.
