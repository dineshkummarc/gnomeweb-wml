DOLLAR

Name

DOLLAR —

Synopsis

DOLLAR(num[,decimals])

Description

DOLLAR returns @num formatted as currency.

* This function is Excel compatible.

Examples

DOLLAR(12345) equals "$12,345.00".

See also

FIXED, TEXT, VALUE.
