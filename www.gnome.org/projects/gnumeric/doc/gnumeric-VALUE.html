VALUE

Name

VALUE —

Synopsis

VALUE(text)

Description

VALUE returns numeric value of @text.

* This function is Excel compatible.

Examples

VALUE("$1,000") equals 1000.

See also

DOLLAR, FIXED, TEXT.
