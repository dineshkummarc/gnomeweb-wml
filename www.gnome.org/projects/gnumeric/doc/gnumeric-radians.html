RADIANS

Name

RADIANS -- 

Synopsis

RADIANS(x)

Description

RADIANS computes the number of radians equivalent to @x degrees.

This function is Excel compatible.

Examples

RADIANS(180) equals 3.14159.

See also

PI, DEGREES.
