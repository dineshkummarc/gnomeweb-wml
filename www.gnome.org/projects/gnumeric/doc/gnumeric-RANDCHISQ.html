RANDCHISQ

Name

RANDCHISQ —

Synopsis

RANDCHISQ(nu)

Description

RANDCHISQ returns a Chi-Square-distributed random number.

Examples

RANDCHISQ(0.5).

See also

RAND, RANDGAMMA.
