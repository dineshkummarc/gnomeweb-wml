IMEXP

Name

IMEXP —

Synopsis

IMEXP(inumber)

Description

IMEXP returns the exponential of a complex number.

  • If inumber is not a valid complex number, IMEXP returns #VALUE! error.
  • This function is Excel compatible.

Examples

IMEXP("2-j") equals 3.992324-6.217676j.
