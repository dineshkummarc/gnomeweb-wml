IMNEG

Name

IMNEG —

Synopsis

IMNEG(inumber)

Description

IMNEG returns the negative of the complex number z (@inumber), where

-z = (-x) + i(-y).

* If @inumber is not a valid complex number, IMNEG returns #VALUE! error.

Examples

IMNEG("1-j") equals -1+j.
