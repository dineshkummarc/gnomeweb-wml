18.1. Authors

Gnumeric was written by Miguel de Icaza <miguel@kernel.org>, Daniel Veillard <Daniel.Veillard@w3c.org>, Tom Dyas <tdyas@romulus.rutgers.edu>, Chris Lahey, Michael Meeks <mmeeks@gnu.org>, Vladimir Vuksan <vuksan@veus.hr>, Takashii Matsuda <matsu@arch.comp.kyutech.ac.jp>, Jukka-Pekka Iivonen <jiivonen@hutcs.cs.hut.fi>, Alan Cox <alan@redhat.com>, Vincent Renardias <vincent@ldsol.com>, Jody Goldberg <jgoldberg@home.com>, Morten Welinder <terra@diku.dk> . To find more information about Gnumeric, please visit the Gnumeric Web page. Please send all comments, suggestions, and bug reports to the GNOME bug tracking database. If you are using GNOME 1.1 or later, you can also use Bug Report Tool (bug-buddy), available in the Utilities submenu of Main Menu, for submitting bug reports.

This manual was written by Thomas Canty (<tommydal@ihug.com.au>), Miguel deIcaza (<miguel@gnu.org>), Adrian Likins (<adrian@gimp.org>) and Jukka-Pekka Iivonen (<jiivonen@hutcs.cs.hut.fi>. Please send all comments and suggestions regarding this manual to the GNOME Documentation Project by sending an email to <docs@gnome.org>. You can also add your comments online by using the GNOME Documentation Status Table.
