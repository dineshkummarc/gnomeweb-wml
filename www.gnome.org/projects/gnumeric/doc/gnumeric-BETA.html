BETA

Name

BETA —

Synopsis

BETA(a,b)

Description

BETA function returns the value of the mathematic beta function extended to all real numbers except 0 and negative integers.

* If @a or @b are 0 or negative integers, BETA returns #NUM! error.

Examples
