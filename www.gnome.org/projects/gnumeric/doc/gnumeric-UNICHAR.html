UNICHAR

Name

UNICHAR —

Synopsis

UNICHAR(x)

Description

UNICHAR returns the unicode character represented by the number @x.

Examples

UNICHAR(65) equals A.

UNICHAR(8232) equals an carriage return error.

See also

CHAR, UNICODE, CODE.
