LOG2

Name

LOG2 —

Synopsis

LOG2(x)

Description

LOG2 computes the base-2 logarithm of @x.

* If @x <= 0, LOG2 returns #NUM! error.

Examples

LOG2(1024) equals 10.

See also

EXP, LOG10, LOG.
