Synopsis

ATL_LAST (tag)

Description

ATL_LAST is a sample implemention of a real time data source. It takes a string tag and monitors the named pipe /tmp/atl for changes to the value of that tag.

  • This is no intended to be generally enabled and is OFF by default.

Examples

ATL_LAST("IBM")
