LEFT

Name

LEFT —

Synopsis

LEFT(text[,num_chars])

Description

LEFT returns the leftmost @num_chars characters or the left character if @num_chars is not specified.

* This function is Excel compatible.

Examples

LEFT("Directory",3) equals "Dir".

See also

MID, RIGHT.
