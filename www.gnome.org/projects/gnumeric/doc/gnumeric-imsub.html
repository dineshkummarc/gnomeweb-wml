IMSUB

Name

IMSUB -- 

Synopsis

IMSUB(inumber,inumber)

Description

IMSUB returns the difference of two complex numbers.

This function is Excel compatible.

Examples

IMSUB("3-j","2+j") equals 1-2j.

See also

IMSUM.
