RANDBERNOULLI

Name

RANDBERNOULLI —

Synopsis

RANDBERNOULLI(p)

Description

RANDBERNOULLI returns a Bernoulli-distributed random number.

* If @p < 0 or @p > 1 RANDBERNOULLI returns #NUM! error.

Examples

RANDBERNOULLI(0.5).

See also

RAND, RANDBETWEEN.
