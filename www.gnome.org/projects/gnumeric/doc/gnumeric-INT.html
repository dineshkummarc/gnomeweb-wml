INT

Name

INT —

Synopsis

INT(a)

Description

INT function returns the largest integer that is not bigger than its argument.

* This function is Excel compatible.

Examples

INT(7.2) equals 7.

INT(-5.5) equals -6.

See also

FLOOR, CEIL, ABS.
