TODAY

Name

TODAY -- 

Synopsis

TODAY()

Description

TODAY returns the serial number for today (the number of days elapsed since the 1st of January of 1900).

This function is Excel compatible.

Examples

TODAY() returns 'Nov 6, 2001' on that particular day.
