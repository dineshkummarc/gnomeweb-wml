IMARCCOT

Name

IMARCCOT —

Synopsis

IMARCCOT(inumber)

Description

IMARCCOT returns the complex arccotangent of the complex number z (@inumber), where

arccot(z) = arctan(1/z).

* If @inumber is not a valid complex number, IMARCCOT returns #VALUE! error.

Examples

IMARCCOT("1+j") equals 0.553574+0.4023595j.

See also

IMARCSEC, IMARCCSC.
