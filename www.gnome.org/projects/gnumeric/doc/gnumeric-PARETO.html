PARETO

Name

PARETO —

Synopsis

PARETO(x,a,b)

Description

PARETO returns the probability density p(x) at @x for a Pareto distribution with exponent @a and scale @b.

Examples

PARETO(0.6,1,2).

See also

RANDPARETO.
