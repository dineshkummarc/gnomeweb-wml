IMCONJUGATE

Name

IMCONJUGATE —

Synopsis

IMCONJUGATE(inumber)

Description

IMCONJUGATE returns the complex conjugate of a complex number.

* If @inumber is not a valid complex number, IMCONJUGATE returns #VALUE! error.

* This function is Excel compatible.

Examples

IMCONJUGATE("1-j") equals 1+j.

See also

IMAGINARY, IMREAL.
