LN

Name

LN -- 

Synopsis

LN(x)

Description

LN returns the natural logarithm of @x. If @x <= 0, LN returns #NUM! error.

This function is Excel compatible.

Examples

LN(7) equals 1.94591.

See also

EXP, LOG2, LOG10.
