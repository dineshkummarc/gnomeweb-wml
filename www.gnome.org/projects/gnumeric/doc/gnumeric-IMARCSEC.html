IMARCSEC

Name

IMARCSEC —

Synopsis

IMARCSEC(inumber)

Description

IMARCSEC returns the complex arcsecant of the complex number z (@inumber), where

arcsec(z) = arccos(1/z).

* If @inumber is not a valid complex number, IMARCSEC returns #VALUE! error.

Examples

IMARCSEC("1+j") equals 1.1185179+0.5306375j.

See also

IMARCCSC, IMARCCOT.
