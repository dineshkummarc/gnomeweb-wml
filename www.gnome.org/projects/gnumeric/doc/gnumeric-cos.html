COS

Name

COS -- 

Synopsis

COS(x)

Description

COS function returns the cosine of @x, where @x is given in radians.

This function is Excel compatible.

Examples

COS(0.5) equals 0.877583.

COS(1) equals 0.540302.

See also

COSH, SIN, SINH, TAN, TANH, RADIANS, DEGREES.
