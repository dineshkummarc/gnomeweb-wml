IMARGUMENT

Name

IMARGUMENT -- 

Synopsis

IMARGUMENT(inumber)

Description

IMARGUMENT returns the argument theta of a complex number.

This function is Excel compatible.

Examples

IMARGUMENT("2-j") equals -0.463647609.
