IMARCCSC

Name

IMARCCSC —

Synopsis

IMARCCSC(inumber)

Description

IMARCCSC returns the complex arccosecant of the complex number z (@inumber), where

arccsc(z) = arcsin(1/z).

* If @inumber is not a valid complex number, IMARCCSC returns #VALUE! error.

Examples

IMARCCSC("1+j") equals 0.45227845-0.5306375j.

See also

IMARCSEC, IMARCCOT.
