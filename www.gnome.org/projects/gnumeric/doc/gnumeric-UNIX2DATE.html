UNIX2DATE

Name

UNIX2DATE —

Synopsis

UNIX2DATE(unixtime)

Description

UNIX2DATE converts a unix time into a spreadsheet date and time.

A unix time is the number of seconds since midnight January 1, 1970.

Examples

See also

NOW, DATE, DATE2UNIX.
