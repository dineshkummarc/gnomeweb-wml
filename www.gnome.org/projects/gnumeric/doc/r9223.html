DELTA

Name

DELTA -- 

Synopsis

DELTA(x[,y])

Description

DELTA function test for numerical equivilance of two arguments returning 1 in equality @y is optional, and defaults to 0.

If either argument is non-numeric returns a #VALUE! error. This function is Excel compatible.

Examples

DELTA(42.99,43) equals 0.

See also

EXACT, GESTEP.
