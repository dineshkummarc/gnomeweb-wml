ROWS

Name

ROWS -- 

Synopsis

ROWS(reference)

Description

ROWS function returns the number of rows in area or array reference.

If @reference is neither an array nor a reference nor a range returns #VALUE!.

Examples

ROWS(H7:I13) equals 7.
