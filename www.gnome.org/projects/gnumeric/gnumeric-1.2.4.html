Gnumeric 1.2.4 aka "A branching we will go" is now available.

With a few more bugs fixed, and some final features enabled for the charting engine Gnumeric has now branched. Version 1.2.3 was not announced due to last minute fixes in xls export. The main extension in this release is the addition of value formats for the axis labels, user defined, auto generated from the source data, or from MS Excel.

Development of 1.3.0 has already begun, so releases of the 1.2.x line will now be bug fix only.
