Downloading Gnumeric

A note on protocols:

The "protocol" column below specifies what protocol the link listed uses. The usual protocols are HTTP (HyperText Transfer Protocol) and FTP (File Transfer Protocol). If Gnumeric is bundled in a distribution, a pseudo-protocol, "Distro" is used to signify that you should download it from that distribution. A link to the distribution's main page is given.

Gnumeric is available from download from the following locations, amongst others:

Using the Dependencies Table

The table is fairly simple to use. Each distribution is listed in the columns, and possible packages are listed in the rows. If the distribution requires a package, the version it requires and a link to get it will be listed, and the background color will be set. If the distribution does not require a package, the cell will be empty, and set to the page's background color. If the package is optional, the cell will have a special color and "(o)". (Note: Gecko-technology browser users won't see the release visualization colors until Mozilla bug 4510 is fixed.)
