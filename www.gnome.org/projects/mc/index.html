GNU Midnight Commander is a Unix file manager and user shell.
Midnight Commander is a file manager with both a console GUI and a GNOME graphical GUI. The GNOME frontend used to be the default file manager in GNOME.

Midnight Commander is still being fully maintained and is an excelent option for people looking for a lightweight filemanager for GNOME or who need a full featured file manager to use in the console or through telnet.
