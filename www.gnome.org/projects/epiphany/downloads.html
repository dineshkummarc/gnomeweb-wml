Building from source

  • Stable:
    • Source: Epiphany 1.2.2
    • Mozilla: Mozilla 1.6 source
    • Dependencies: GTK+ 2.3/2.4, Mozilla with gtk2 support
  • Previous Stable Series:
    • Source: Epiphany 1.0.8
    • Mozilla: Mozilla 1.6 source
    • Dependencies: GTK+ 2.2, Mozilla with gtk2 support
  • Development: ---
  • We recommend to use Mozilla 1.6. Other Mozilla versions known to work: Mozilla 1.4, 1.5, CVS.

Suggested way to compile Mozilla

./configure --prefix=YOUR_GNOME_PREFIX --enable-default-toolkit=gtk2 --disable-mailnews --disable-ldap --disable-debug --disable-tests --enable-optimize --enable-crypto --enable-xft --with-system-zlib
make
make install
