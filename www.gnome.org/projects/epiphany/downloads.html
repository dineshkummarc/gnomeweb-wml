Building from source

  • Stable:
    • Source: Epiphany 1.4.1
    • Mozilla: Mozilla 1.7.3 source
    • Dependencies: GTK+ 2.4, Mozilla 1.7.x with gtk2 support
  • Previous Stable Series:
    • Source: Epiphany 1.2.9
    • Mozilla: Mozilla 1.7.3 source
    • Dependencies: GTK+ 2.2, Mozilla 1.4, 1.4.x, 1.5, 1.6, 1.7.x with gtk2 support
    • Note: to compile Epiphany 1.2.9 with Mozilla 1.7.3, you need to configure it with --with-mozilla-version=1.7.2 argument to the configure script.
  • We recommend to use Mozilla 1.7.3.

Suggested way to compile Mozilla

./configure --prefix=YOUR_GNOME_PREFIX --enable-default-toolkit=gtk2 --disable-mailnews --disable-ldap --disable-debug --disable-tests --enable-optimize --enable-crypto --enable-xft --with-system-zlib
make
make install
