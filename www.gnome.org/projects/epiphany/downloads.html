Building from source

Stable:

  • Source: Epiphany 1.0.5
  • Mozilla: Mozilla 1.5 source
  • Dependencies: GNOME 2.2, Mozilla with gtk2 support

Development:

  • Source: Epiphany 1.1.0
  • Mozilla: Mozilla 1.5 source
  • Dependencies: GTK+ 2.3 (to be released soon), Mozilla with gtk2 support

Other Mozilla versions known to work:

  • Mozilla 1.4, 1.4.1, or 1.6a
  • Cvs Mozilla until build 20030930

Suggested way to compile mozilla

./configure --prefix=YOUR_GNOME_PREFIX
	--enable-default-toolkit=gtk2 --disable-mailnews --disable-ldap
	--disable-debug --enable-optimize --disable-tests --enable-crypto
	--enable-xft --with-system-zlib
make
make install
