4.1. register_module

Epiphany will look for a register_module function. Create a file for it: call it foo.c:

/* Header/License/Copyright information */

#include "ephy-foo-extension.h"
#include "ephy-debug.h"

#include <gmodule.h>

G_MODULE_EXPORT GType
register_module (GTypeModule *module)
{
  LOG ("Registering EphyFooExtension")

  return ephy_foo_extension_register_type (module);
}
Tip

LOG is included in ephy-debug.h. If you use --enable-debug while running configure or autogen.sh from within the ephyplugins/ directory, the messages will be printed with g_print, which uses a printf-like syntax.
Warning

Epiphany extensions could easily be classified as derivative works of Epiphany, which means they require a GPL license. Copy the header comment from an existing extension.
