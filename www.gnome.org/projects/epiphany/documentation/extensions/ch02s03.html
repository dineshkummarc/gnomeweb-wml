2.3. Required Source Code

You will need the source code to Epiphany. It is available at http://www.gnome.org/projects/epiphany. It's a good idea to code an extension for the newest version of Epiphany available; from CVS is best.

The existing GNOME Web Browser Extensions source code has many working (and non-working) examples.
