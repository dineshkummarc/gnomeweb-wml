Connecting Signals

GLib has a very flexible signals system. Use it. The impl_attach_window function has an EphyWindow argument. As an example, the tab_added signal will be fired whenever a tab is added to that window. Put the following in impl_attach_window to hook the tab_added signal to your own callback:

Example 4.1. Connecting the tab_added signal 

GtkWidget *notebook;

	notebook = ephy_window_get_notebook (window);

	g_signal_connect_after (notebook, "tab_added", G_CALLBACK (tab_added_cb), extension);
	g_signal_connect_after (notebook, "tab_removed", G_CALLBACK (tab_removed_cb), extension);

tab_added_cb can connect signals of its own. Of particular interest are the EphyEmbed signals (prefixed by ge_). These signals are listed in epiphany/embed/ephy-embed.c. For example, to call a function when the browser changes to a new location:

Example 4.2. Listening to location changes 

static void
location_cb (EphyEmbed *embed, char *address, EphyTab *tab)
{
	/* Actions to perform on location change */
}

static void
tab_added_cb (GtkWidget *notebook, EphyTab *tab, EphyFooExtension *extension)
{
	EphyEmbed *embed;

	g_return_if_fail (EPHY_IS_TAB (tab));

	embed = ephy_tab_get_embed (tab);
	g_return_if_fail (EPHY_IS_EMBED (embed));

	g_signal_connect (G_OBJECT (embed), "ge_location", G_CALLBACK (location_cb), tab);
}

Warning

Free all memory and disconnect all signals when they're not needed. In the case of this ge_location signal connection, disconnect the signal when the tab is removed:

static void
tab_removed_cb (GtkWidget *notebook, EphyTab *tab, EphyFooExtension *extension)
{
	EphyEmbed *embed;

	g_return_if_fail (EPHY_IS_TAB (tab));

	embed = ephy_tab_get_embed (tab);
	g_return_if_fail (EPHY_IS_EMBED (embed));

	g_signal_handlers_disconnect_by_func
		(embed, G_CALLBACK (location_cb), tab);
}

Likewise, clean up in impl_detach_window. In our example:

	GtkWidget *notebook;

	notebook = ephy_window_get_notebook (window);

	g_signal_handlers_disconnect_by_func
		(notebook, G_CALLBACK (tab_added_cb), extension);
	g_signal_handlers_disconnect_by_func
		(notebook, G_CALLBACK (tab_removed_cb), extension);
