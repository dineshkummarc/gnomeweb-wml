4.4. The Extension Interface

Besides the basic GObject functions, the Epiphany extensions manager will only call two functions directly: the attach_window and detach_window functions in its class. Write skeleton functions, and your extension should actually compile and run:

static void
impl_attach_window (EphyExtension *extension, EphyWindow *window)
{
	LOG ("EphyFooExtension attach_window")
}

static void
impl_detach_window (EphyExtension *extension, EphyWindow *window)
{
	LOG ("EphyFooExtension detach_window")
}

static void
ephy_foo_extension_iface_init (EphyExtensionClass *iface)
{
	iface->attach_window = impl_attach_window;
	iface->detach_window = impl_detach_window;
}

You're now ready to write code. Your creativity should begin in impl_attach_window and extend beyond limits. The next sections just contain ideas to spark your creativity.
