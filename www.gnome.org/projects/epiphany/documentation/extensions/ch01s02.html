1.2. Document Scope

This document is not a C programming tutorial. It is also not a guide to the Epiphany source code. It is just a basic guide to begin writing an extension.

The document is written under the assumption that the reader wants to write an extension. The document will be an aid, but it does not replace reading actual source code from existing extensions. Many extensions are available at http://savannah.nongnu.org/projects/ephyplugins.
