4.6. Translation

If you need to translate some strings (if your extension prints or displays any messages besides debug messages), a few steps must be followed:

  • Add the following line to the source files needing translation and to the file containing register_module (register_module):

    #include <glib/gi18n-lib.h>

  • Add the following to your register_module function:

    #ifdef ENABLE_NLS
	/* Initialize the i18n stuff */
	bindtextdomain (GETTEXT_PACKAGE, EPHY_EXTENSIONS_LOCALEDIR);
	bind_textdomain_codeset (GETTEXT_PACKAGE, "UTF-8");
#endif /* ENABLE_NLS */

  • For every string needing translation, use the N_, _ or ngettext functions. For example:

    g_print (_("This is a foobar. It says %s."), message);
g_print (ngettext ("%d cow jumped over the moon", "%d cows jumped over the moon", n), n);

  • Add the filenames with translatable strings to ephyplugins/po/POTFILES.in.
