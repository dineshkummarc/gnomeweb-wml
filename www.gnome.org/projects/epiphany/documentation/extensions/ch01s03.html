Getting Help

For more help writing Epiphany Extensions, log on to #epiphany on irc.gnome.org. You can also use the regular Epiphany Mailing List.

As with all software projects' channels of communication, please do not ask stupid questions. If you've never read How to Ask Questions the Smart Way by Eric S. Raymond, read it now. It's a big pile of common sense, and it applies to virtually every open-source software project in existence.

If you are a proficient hacker and don't understand some part of this document, that's a bug in the document; please email the author.
