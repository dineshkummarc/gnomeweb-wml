Anjuta2

Lead Developer: JP Rosevear and Naba Kumar

Anjuta2 is an Integrated Development Environment for the GNOME platform. It currently provides an editor with syntax hilighting and a rudimentary project management system.

The new development version has been almost completely rewritten, and is extensible using the Bonobo component model. There is a short roadmap of future development available.

Anjuta2 was originally written by Steffen Kern.

Anjuta2 is meant to aid users with development work. It provides an editor with syntax hilightlighting, a rudimentary project/build system and an interface to the gdb debugger. The functionality can be extended with plugins that create various tools.

(Note: if using Ximian packages, be sure to get the development packages as well, i.e.: libbonobo-dev, and watch out for old versions of libraries lying around that could cause conflicts.)
