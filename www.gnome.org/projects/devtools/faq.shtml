GNOME Devtools FAQ

  1. Can my development tool become part of the GNOME devtools suite?
  2. What is the criteria for which applications gets listed as a component of the GNOME development tools suite and which components gets listed in the related projects listing?


  1. Can my development tool become part of the GNOME devtools suite?
    Yes, all toos that are made using the GNOME development plattform and which integrates with the wider variety of development tools can become part of the GNOME devtools suite.
  2. What is the criteria for which applications gets listed as a component of the GNOME development tools suite and which components gets listed in the related projects listing?
    All applications that aims at interoperating with the other tools on the GNOME devtools page gets listed on the main page. Other GNOME applications which has a more stand-alone approach gets listed on the related projects page.
