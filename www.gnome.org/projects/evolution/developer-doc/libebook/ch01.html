Evolution API Reference: libebook, the Addressbook Library

EBook — The primary Evolution addressbook client object.
EBookListener — Listens for call responses and event notifications from EBook objects.
e-book-query — Handles queries of addressboook data.
e-book-types — Client-side GOBject which exposes the Evolution:BookListener interface.
EBookView — Represents a search against an EBook.
EBookViewListener — Listens for call responses and event notifications from EBookView objects.
EVCard — Contact data in the VCard format.
EContact — An individual contact in an addressbook.
EDestination
e-address-western — A physical address in the format used in the Western hemisphere.
