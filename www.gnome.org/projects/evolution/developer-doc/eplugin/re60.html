Name

enum _e_menu_t — Menu item type.

Synopsis

enum _e_menu_t {
  E_MENU_ITEM,
  E_MENU_TOGGLE,
  E_MENU_RADIO,
  E_MENU_TYPE_MASK,
  E_MENU_ACTIVE
};

Constants

E_MENU_ITEM

Normal menu item.

E_MENU_TOGGLE

Toggle menu item.

E_MENU_RADIO

unimplemented.

E_MENU_TYPE_MASK

Mask used to separate item type from option bits.

E_MENU_ACTIVE

Whether a toggle item is active.

Description

The type of menu items which are supported by the menu system.
