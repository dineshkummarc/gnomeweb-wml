Name

e_config_page_get —

Synopsis

GtkWidget * e_config_page_get (ec,  
 pageid); 
EConfig *  ec;
const char *  pageid;

Arguments

ec

pageid

The path of the page item.

Description

Retrieve the page widget corresponding to pageid.

Return value

The page widget. It will be the root GtkNotebook container or the GnomeDruidPage object.
