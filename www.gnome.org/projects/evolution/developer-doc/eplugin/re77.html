e_menu_class_remove_factory
Prev Chapter 10.  EMenu  Next

Name

e_menu_class_remove_factory —

Synopsis

void e_menu_class_remove_factory (klass,  
 f); 
EMenuClass *  klass;
EMenuFactory *  f;

Arguments

klass

Class on which the factory was originally added.

f

Factory handle.

Description

Remove a popup factory. This must only be called once, and must only be called using a valid factory handle f. After this call, f is undefined.
