Name

e_config_abort —

Synopsis

void e_config_abort (ec);
EConfig * ec;

Arguments

ec

Description

Signify that the stateful configuration changes must be discarded to all listeners. This is used by self-driven druid or notebook, or may be used by code using the widget directly.
