EPlugins

Overview of EPlugin

The EPlugin system (introduced in Evolution 2.2) exists to provide modular extensions to Evolution by providing a simple, practical, language-neutral approach to extending the Evolution groupware suite. Thanks to EPlugins, users of Evolution can have extended functionality and developers can easily enhance Evolution with minimal effort.

It is not a scripting language, although it could be used to add a scripting language to Evolution. It is a framework of helper utilities to aid application development and simultaneously provide hooks whereby external code can access internal features — both user interface and behavioural.

EPlugin's goals

  • Provide a language independent invocation mechanism
  • Allow extension of parts of the user interface and processing elements
  • Require minimal extra or foreign code to implement in the core application
  • Require minimal interface code to implement the extensions
  • Not to impact performance or increase resource usage unduly
  • Versioning
  • The ability to extended itself fairly easily

Available EPlugins

Currently, EPlugins are included in Evolution's source.

Strictly Reference EPlugins

copy-tool

A tool to extend various clipboard operations
A test plugin which demonstrates a popup menu plugin which lets you copy things to the clipboard

prefer-plain

Prefer plain-text plugin
A test plugin which demonstrates a formatter plugin which lets you choose to disable HTML mails

folder-unsubscribe

Bob's Wonder Extender
Extends wonders

Totally Useful EPlugins

bbdb

BBDB Functionality for Evolution
Automatically tracks contacts

mail-to-task

Convert a mail message into a task
A plugin which allows the creation of tasks from the contents of a mail message

save-attachments

Save attachments plugin
A plugin for saving all attachments or parts of a message at once.

save-calendar

Save Selected Calendar or Tasks List
Saves selected calendar or tasks list to disk

select-one-source

Select one source
Selects a single calendar or task source for viewing

subject-thread

Subject Threading
Indicates if threading of messages should fall back to subject

audio-inline

Audio inline plugin
A formatter plugin which displays audio attachments inline and allows you to play them directly from evolution

mark-calendar-offline

Mark calendar offline
Marks the selected calendar for offline viewing
