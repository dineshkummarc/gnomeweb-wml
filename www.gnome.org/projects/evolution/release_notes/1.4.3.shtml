Evolution 1.4.3 Release Notes

Ximian Evolution 1.4.3 is now available. This update includes bug fixes as a result of community and customer feedback received since version 1.4.0.

Ximian Evolution 1.4.1 and 1.4.2 were not publicly released due to issues that have since been fixed.

Supported Systems

Evolution 1.4.3 is supported on the following operating systems:

  • Mandrake Linux 9.1
  • Red Hat Linux 7.3, 8.0 and 9
  • Suse Linux 8.2

Support for Solaris 8 will be released shortly.

Known Issues

  • LDAP support is not currently functional under SuSE 8.2. An updated package to fix this problem will be available in the near future.
