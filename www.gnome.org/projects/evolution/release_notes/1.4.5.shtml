Evolution 1.4.5 Release Notes

Ximian Evolution 1.4.5 is now available. This update includes bug fixes as a result of community and customer feedback received since version 1.4.4. This release also includes an update for Ximian Connector.

Supported Systems

Evolution 1.4.5 is supported on the following operating systems:

  • Mandrake Linux 9.1
  • Red Hat Linux 7.3, 8.0 and 9
  • Suse Linux 8.2
  • Solaris 8

Exchange Connector
  • The contents of calendar folders are now cached to local disk between sessions, which will greatly improve performance for users with very full calendars. [#39601]
  • Exchange 2003's "Forms-based authentication" mode is now supported.
  • Improved the account autoconfiguration wizard in several ways:
    • Gives an explicit error message if the server is an Exchange 5.5 server, rather than just a generic failure message [#41538]
    • Gives an error message if the server has been configured with the IIS Lockdown Wizard in a way that will prevent Connector from working correctly. [#46057]
    • Tries appending "..change" to the given URL/hostname if no path was provided and trying to connect with no path failed [#45198]
    • Gives the correct error message now if you give a non-existent username [#35930]
  • Fixed a bug that could cause "new mail" notifications from the server to be dropped sometimes. [#47702]
  • Fixed two obscure crashers [#47754, #47765]
