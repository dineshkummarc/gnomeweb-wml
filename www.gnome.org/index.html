What is GNOME?

The GNOME project offers an easy to understand, intuitive and attractive desktop for GNU/Linux or other UNIX based operating systems. Learn more about GNOME as a whole, or get help and support from the GNOME community.

GNOME also provides an extensive development platform and framework for building applications that integrate into the desktop. Find out more about developing with GNOME, and how you can get involved.
