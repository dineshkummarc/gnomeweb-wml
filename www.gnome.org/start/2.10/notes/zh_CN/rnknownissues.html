已知的问题

所有软件在其发布之时包含了了一些开发者已知的 BUG，但因为各种原因，未能在发布之前修正。自由软件在这方面和专利的软件没有什么不同，但是有一点除外，我们会把这些问题告诉用户。

我们也鼓励用户去报告 BUG，以便可以及时的修正它们。在 GNOME 中最好办法就是使用 简单 BUG 指南。它将引导你完成填写一个合格的 BUG 报告，确保 BUG 被正确地标识。如果你已经非常熟悉‘简单模式’中所有内容，那么你也可以访问传统的 BUG 表单。关于已经被报告过的 BUG 的详细信息你可以在我们的 Bugzilla 上查看。GNOME 2.10 中最主要的 BUGS 如下：

已知问题的清单

  • Mozilla/Firefox 新窗口焦点：在你启动 Mozilla 或者 Firefox 时，如果已经有另一个 Mozilla 或者 Firefox 窗口已经开启，新的窗口出现后可能会因为没有获取到焦点而显示在有焦点的窗口后面。GNOME 的基于 Mozilla 的 Epiphany 浏览器没有这样的问题。Mozilla 黑客们正在努力的工作，以期在未来的 Mozilla 和 Firefox 版本解决这个问题。

  • 会话启动时的短暂挂住 ：有会话管理问题的程序会使得登录后的欢迎屏幕挂住几分钟。缺省的会话不包含这样的程序，只有在你注销时保存了会话就会出现这样的问题。如果你遇到了这样的问题，只需要等到登录完成后，关闭类似的有问题的程序后，再保存一次会话。
