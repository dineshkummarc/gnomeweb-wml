GNOME 2.10 中有什么新变化

桌面

文件管理器

GNOME 2.10 中的文件管理器，也就是 Nautilus，更快更稳定了。这要感谢其内部的构架改变。所有这些使得软件开发者在以后更轻松地对这个文件管理器进行维护和扩展。

与其他程序，比如 Mozilla 和 Firefox 之间的拖放更加平滑。

另外，还有许多小变化，显示出 GNOME 对细节的关注。比如，

  • 路径按钮也更像一个按钮了。

  • 在创建新文件时，会自动进入重命名状态。

  • 使用键盘导航来打开上一级文件夹时(Alt-Up)，当前的文件夹将被选中。

图 1. 简洁而强大的 Nautilus 文件管理器

GNOME 2.10 提供一个新的预安装的背景图片和图案选集，您将拥有自己的桌面。

图 2. 一个简单而且具有艺术感的背景选集

窗口行为

GNOME 2.10 有个可能被您所忽略的地方，那就是实现了另外一个跨桌面(cross-desktop)的 Freedesktop.org 标准。

过去，当您正一个应用程序中输入某些内容时，您的即时信息突然弹出一个来自您的朋友的聊天请求，那些单词就会被输入到聊天窗口中去。想象如果当时您正在输入密码的话...这种情形永远也不会发生在 GNOME 2.10 上。

另外，如果一个应用程序启动时要耗去较长的时间，在它最终打开自己的窗口之前不会打断您的工作。

应用程序

视频播放器

现在 GNOME 2.10 包含了一个流行的 Totem 播放器，它使用了 GNOME 的 Gstreamer 多媒体架构。Totem 提供 GNOME 用户期望的直观的用户界面，用自动化的硬件检测来替换复杂的设定。

图 3. 视频播放器

音频抓取

相当可爱的 Sound Juicer“CD 提取器”能轻松地从 CD 中抓取音频，然后在您的 PC 或便携音乐播放器中播放音频。由于可以自动下载歌曲标题，所以您无须点击多于两个的按钮。

图 4. 音频抓取器

网络浏览器

GNOME 的“Epiphany”网络浏览器是基于 Mozilla 同时也与 GNOME 桌面完整的整合在一起。在 GNOME 2.10 的改进包括

  • 经过改进的全屏模式。

  • 地址条现在会显示网站是否是安全的，并显示相关图标。

  • 书签可以导出。

  • 一个做为额外功能的扩展管理器。

图 5. 网络浏览器

Evolution

GNOME 集成的电子邮件和群件客户端，Evolution 支持传统的邮件设定，也包括 Novell Groupwise 和 Microsoft Exchange。通过 Evolution ，您可以读、写、和管理您的电子邮件、通信簿、和日历事件。

在 GNOME 2.10 中，如果您使用 IMAP、LDAP、WebCal、Groupwise、或 Exchange， 在离线状态下处理电子邮件，日日历，和联系簿也变得更加轻松。您所做的变更在您重新在线时会自动同步。

这个新版本提供了其他一些改进：

  • 文件可以附到事件中。

  • 在循环事件中可放置异常。

  • 日历包含天气信息(仅限于美国)。

图 6. 每天的天气预报

更多的新特性：

  • 支持 Groupwise 共享文件夹和发送选项。

  • 支持 Exchange 文件夹大小和密码期限警告。

  • 对于从右到左顺序的语言，电子邮件用户界面可以镜像过来。

图 7. 电子邮件客户端

GnomeMeeting

GnomeMeeting 允许用户通过 internet 电话(VoIP)和视频会议互相看见、交谈。在 GNOME 2.10 中，地址簿和 Evolution 电子邮件客户端是共享的，这样您就无须在多个地方输入联系信息了。

现在您甚至可以在局域网浏览查找其他的 GnomeMeeting 用户，而无须先行了解他们的联系细节。同时，您可以看见自己的视频输出与交流伙伴并排在那里，这样您看见的和他们看见的一模一样。

图 8. Internet 电话(VoIP)和视频会议

控制中心

键盘布局

通过键盘控制面板来选择键盘布局变得更加容易了。只要点击添加就可以在国际化键盘布局列表中选择。在做选择之时就可以预览到键盘布局。布局选项部分也重新组织成更简单。

图 9. 键盘设置-布局

面板项目

GNOME 2.10 包括重要的面板项目的(也被称为小程序，Applet)改进，提供新特性和更好地与桌面的其它部分整合。

最显著的变化是去掉了动作面板菜单，添加了位置和桌面菜单。这使得你可以更加快捷地访问你最常使用的项目，而无须在子菜单中去寻找。比如，位置菜单列出了您的主文件夹，您的计算机，网络服务器，还有搜索。在桌面菜单，会发现您的设置和管理控制面板，注销，和帮助。

全部面板菜单是由跨平台的 Freedesktop.org 菜单规范所定义的，第三方软件甚至无须了解 GNOME 就可以把自己添加到应用程序菜单中。

同样，当面板自身是透明的时候，大多数的小程序(Applets)能实现局部透明了。

整合的 Moden 控制

GNOME 2.10 引入了一个新的小程序来控制 Modem，这个小程序已经整合入到了 GNOME 系统工具中。

图 10. 新的 modem 控制 applet

面板垃圾桶

GNOME 2.10 为您的面板带来一个可选的垃圾桶。就象桌面上的垃圾桶一样，可以拖放文件和其他项目到垃圾桶中，而无须最小化打开的窗口。

图 11. 一个面板上的垃圾桶

控制挂载的卷标

GNOME 2.8 允许您轻松挂载数字媒介，比如 CD, DVD, 和 USB 存贮棒。现在卷标挂载器 applet 助您使用这些卷标，无须在桌面上困难地寻找在打开窗口之下的相关图标。这个 applet 显示当前已挂载(或未挂载)的卷标，让您挂载，弹出，或者在文件管理器中打开它们。

图 12. 驱动挂载

得到的不仅仅是天气

天气监视器现在可以现实比以前更多的地区了，在适当的地方，地区名称也翻译成非英语语言。另外，预报包括了大多数地方的日出日落计算。

图 13. 天气报告

声音控制

面板上的一个新混音器(包含桌上的面一个新外观的混音器)从你声音硬件取得大多数属性。通过 GStreamer 多媒体构架，如今的混音器比以前更加简单和有效。在通过面板改变 GNOME 的音量时，所有使用相同构件的 Totem，Sound Juicer, GNOME CD 播放器和其他的第三方应用程序的音量也会改变。

图 14. 面板上的全新混音器

做记录

流行的即时帖 applet 在桌面上现在更加容易添加 PostIt 类型的记录，这得益于更加简单的用户界面和对许多问题的修正。比如，即使是在不同的显示器上，即时帖也会停留在正确的工作区上。

即时帖处于其他窗口之上，这样您就不会失去它们。要隐藏可以使用小程序的右键菜单。

图 15. 即时帖

特别为笔记本提供

GNOME 包含了一个监视器来监视笔记本的处理器速度。CPU 频率监视器查看变速的处理器(比如那些拥有Intel Speedstep或AMD PowerNow的处理器)以显示它们的当前状态。如果没有开启自动调整频率功能，您也可以自己选择不同的速度。

图 16. 监视器 CPU 频率变化

已移除的小程序

在 GNOME 2.10 中有三个小程序被去除掉了，取而代之的在多数情况下更佳的选择。

CD 播放器面板 applet 已经被去除，所以即使你把 CD 放入了驱动器，您也不能再看到CD播放器出现在面板上。做为替代方案，在插入 CD 会启动 GNOME CD 播放器，然后可以从面板上的通知区域图标来控制它。

因为包含在 GNOME 2.8 的网络监视器小程序包含了无线支持，所以专门的无线小程序被移除了。

因为无人维护和不安全，邮件监视器被除去。我们希望在将来有个与我们的 Eolution 邮件客户端整合在一起的解决方案。如果没有使用 Evolution，您可以使用一个第三方的应用程序比如 mailnotify。

实用工具

GNOME 实用工具已经做了一些改进，比如：

文本编辑器

现在 GNOME 文本编辑器可以突出显示当前行，并且在编辑程序源代码时会突出显示匹配的花括号，语法检查也得到改进。只在真正完成单词输入时才对拼写错误的词进行突出显示，并支持许多其他语言，也可以自动定义。

大家会很乐意看到，在所以变动都被取消后它不再建议保存变动。还有就是它的启动速度比之前的版本要快。

图 17. 文本编辑器

归档管理器

现在 GNOME 档案管理器可以打开更多类型的档案，包括 AR，Debian，和 7-zip 多种档案，以及密码保护的RAR档案。 如果文件管理器是单击设定，档案管理器也使用单击。

图 18. 归档管理器

词典

现在 GNOME 词典在您不确定如何拼写时会给出建议。解释也包含词典网站的链接，同时，这些解释也可以打印出来。

图 19. 词典

软盘格式化工具

现在 GNOME 软盘格式工具开始支持可移动驱动器，比如 USB 驱动器，这是通过利用 freedesktop.org 的标准 HAL 系统实现的。

图 20. 软盘格式化工具

系统管理

GNOME 2.10 为系统管理提供了某些新特性

系统工具

系统工具允许您配置您的系统时钟和网络连接，也可以管理系统中的用户和群组。目前，这些系统工具比较适合单机，而非联网计算机。

GNOME 2.10 的某些改进，包括

  • Networking: Improved wireless support (WEP and ESSID detection), and ISDN connections.

  • 用户和群组：更易编辑群组的编号。

  • 在适当的地方，改动会即时生效。

图 21. 时钟，网络和用户/群组工具

日志查看器

GNOME 日志查看器增加了一个十分便利的查找功能，增强的了的监视模式。另外，它现在可以

  • 在分离的多个窗口中打开多个日志。

  • 打开归档后的日志和与网络相关的日志。

  • 复制日志到剪贴板中。

图 22. 日志查看器

游戏

GNOME 助您实现您的目标，但是有时您的目标可能令您心烦意乱。这时 GNOME 游戏为您送上乐趣。

Same GNOME

Same GNOME 游戏有个重要的新版本，拥有更好的图形，超大的面板尺寸，和在您改变主意之时用到的撤销功能。

图 23. Same GNOME

Nibbles

我们的“Just Works” 理论在 Nibbles 游戏中也得到了体现。通过使用与 GnomeMeeting 相同的 ZeroConf技术，现在您可以浏览局域网寻找他人来玩对战。

图 24. 加入网络游戏

平台改进

GNOME 2.10 开发平台为第三方软件开发者以及 GNOME 桌面自身提供了一个稳定的基石。在维护向后兼容性和 API的稳定的同时，GNOME 2.10 对一些 API 做了改进。这时，具体的来说有

  • GTK+ 2.6 拥有新的单元(cell)渲染器，新的按钮，图标视图，和新的关于对话框。

  • glib 2.6 有个更加易用的命令行解释 API。

  • 通过 gnome-python，平台绑定为 GNOME 开发平台提供了了全套的 Python APIs。还有 C++, Java, Perl, 和 Python 绑定，这些是通过 gtkmm, java-gnome, gtk2-perl, 和 pygtk 众多项目得到的。
