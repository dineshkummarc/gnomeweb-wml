Pemasangan GNOME 2.10

Untuk GNOME 2.10 kami memliki LiveCD, tersedia melalui www.gnome.org. LiveCD memudahkan anda untuk mencoba desktop GNOME secara menyeluruh pada linux tanpa memasang apapun pada hard disk anda. Ini adalah cara paling baik bagi anda untuk mengetahui apa saja yang baru.

Untuk penggunaan nyata, kami menyarankan bahwa anda memasang paket resmi, seperti pada distribusi linux anda. Vendors sepertinya akan memaketkan GNOME 2.10 dengan sangat cepat, dan untuk melepas terbitan baru segera dengan mengikut sertakan GNOME 2.10.

Bagaimanapun, jika anda ingin membangun GNOME dari kode sumber, untuk mencoba versi yang paling terakhir, untuk menyediakan umpan balik dan peningkatan, kami menyarankan anda untuk membangun suatu alat, seperti utilitas GARNOME, untuk membangun dari terbitan tarball dan jhbuild untuk membangun dari CVS.
