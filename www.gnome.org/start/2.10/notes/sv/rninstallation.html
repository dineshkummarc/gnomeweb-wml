Installation av GNOME 2.10

För GNOME 2.10 finns det nu en LiveCD som kan hämtas från www.gnome.org. Denna cd-skiva möjliggör för dig att prova GNOME-skrivbordet utan att installera någonting på din hårddisk. Detta är det bästa sättet att själv se vad som är nytt.

För verklig användning rekommenderar vi att du installerar officiella paket, som till exempel de för din Linux-distribution. Distributörer kommer sannolikt ganska snart att paketera GNOME 2.10 och släppa nya versioner som innehåller GNOME 2.10.

Om du ändå vill bygga GNOME från källkod för att testa de senaste versionerna och tillhandahålla återkoppling och förbättringar rekommenderar vi ett byggverktyg, som till exempel verktyget GARNOME för byggande från släppta tarbollar, och jhbuild för byggande från CVS.
