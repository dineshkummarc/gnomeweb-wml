Concordância com padrões

O GNOME trabalha lado a lado com grupos como o freedesktop.org. O suporte a padrões é um grande adicional para os usuários do GNOME. O suporte a interoperabilidade melhora a experiência do usuário na medida que permite que aplicações para o GNOME, para o KDE e para outros ambientes funcionem juntos mais facilmente. E ao seguir especificações abertas, o GNOME ajuda a garantir que os dados do usuário não estão presos num formato proprietário.

Os desenvolvedores do GNOME estão trabalhando pesado com outros membros da comunidade de software livre através do Freedesktop.org no desenvolvimento de padrões para permitir interoperabilidade. Esses padrões incluem especificações de banco de dados MIME compartilhado, de temas de ícones, de arquivos recentes, de menus, de entradas da área de trabalho, de gerenciamento de miniaturas e da bandeja do sistema. Além disso, o GNOME suporta CORBA, XML, Xdnd, EWMH, XEMBED, XSETTINGS e XSMP.
