Встановлення GNOME 2.10

Для GNOME 2.10 є LiveCD, що доступний на www.gnome.org. LiveCD дозволяє випробувати повноцінне середовище GNOME на Linux не встановлюючи його на жорсткий диск. Це найкращий шлях для огляду новинок.

Для фактичного використання, ми рекомендуємо встановлювати офіціальні пакети, наприклад з дистрибутивів Linux. Розробники досить швидко створюють пакети для GNOME 2.10, та скоро випустять нові версії, що включатимуть GNOME 2.10.

Проте, якщо ви бажаєте зібрати GNOME з вихідного коду, щоб перевірити останню версію для зворотного зв'язку чи вдосконалень, ми рекомендуємо систему збирання: наприклад утиліту GARNOME, для збирання з tar-архівів, та jhbuild, для збирання з CVS.
